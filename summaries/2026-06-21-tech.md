  📎 來源：[Simon Willison](https://simonwillison.net/2026/Jun/19/sean-lynch/#atom-everything)

• 🔥8 **CrewAI 1.14.8a2 加入 single agent action 與 Flow CEL 驗證** — CrewAI alpha release 新增 Flow definition 的 single agent action，並在 definition load time 驗證 Flow CEL expression，減少執行到一半才發現流程條件錯誤的機率。批評視角是，agent action 越容易被嵌進 workflow，越需要明確的 tool permission、dry-run 與 rollback。
  📎 來源：[crewAIInc/crewAI](https://github.com/crewAIInc/crewAI/releases/tag/1.14.8a2)

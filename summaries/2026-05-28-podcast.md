第85集。

早安，今天是5月28日。今天這集，我想先從兩個壓力點講起：一個是剛果 Ebola 疫情跨過邊境，讓公共衛生、戰爭和國家管制同時撞在一起；另一個是 AI 和 stablecoin 都在往真正的日常基礎設施移動，所以它們的問題也不再只是工程師或幣圈自己的問題。先進世界新聞。

今天最需要停下來看的，是剛果東部的 Ebola 疫情。到5月27日，剛果疑似病例已經超過一千例，死亡至少兩百二十人，烏干達也因為境內出現病例，宣布立即關閉與剛果的邊境。這件事的關鍵不是數字變大而已，而是它已經從單一國家的疫情，變成跨境公共衛生事件。也就是說，過去幾天我們看到的醫院被衝擊、疑似病患逃離、醫護缺防護裝備，現在會跟邊境管制、旅客隔離、國際援助和地方戰事綁在一起。

在剛果東部，疫情不是發生在一個治理完整、交通清楚、醫療資源充足的地方。AP 在 Bunia 的報導很具體：一個一萬人的流離失所者營地，只有一個洗手站和一支紅外線體溫計。這代表最基本的防線，像洗手、隔離、追蹤接觸者，都不是理所當然。WHO 也警告，衝突和人口移動會讓疫情跑得比反應更快。你可以把它想成兩條曲線，一條是病毒傳播，一條是政府和援助組織把人、物資、信任送到現場的速度；現在前者明顯跑在後者前面。

更麻煩的是信任。Le Monde 提到，部分社區對醫療工作者有深刻懷疑，甚至有人相信疾病是外來者創造的。這種說法在危機時很常見，但它不是單純的迷信問題，而是多年戰爭、外部干預、政府缺席留下來的政治後果。當醫療團隊不能被信任，隔離就會被看成拘禁，安全埋葬就會被看成奪走親人，邊境關閉也可能把人推向更隱蔽的路線。烏干達關邊境可以爭取時間，但如果沒有同步的檢測、溝通和照護，它只能延後問題，不能解決問題。

這就是今天世界新聞裡真正的深水區：公共衛生不是只靠疫苗或醫院，它靠的是治理能力。剛果這次疫情如果繼續擴大，印度和非洲聯盟延後原本5月28日要在新德里舉行的 India-Africa Forum Summit，就只是第一個外交外溢。之後可能出現的是更多旅行限制、物資調度壓力，還有國際組織必須在戰區裡重建最低限度的信任。

同一個「壓力外溢」也在中東發生。美國和伊朗的停戰談判還沒真正落地，美軍卻又對伊朗飛彈發射點和疑似布雷船隻發動所謂自衛性打擊。華府的說法是保護艦隊和航道安全，德黑蘭的說法是這違反停火。這裡最重要的新事實，是談判沒有中止，但談判桌旁邊仍然有軍事行動。當荷莫茲海峽的通行、伊朗被凍結資金、核問題和制裁還在同一包協議裡，任何一個小規模交火都會被市場和區域盟友放大解讀。

黎巴嫩那邊也一樣。以色列在 Eid al-Adha 期間要求黎巴嫩南部居民撤離，前一天以軍和 Hezbollah 已經在戰略河段交火，而且以軍推進到更北的位置。這讓華府安排的黎巴嫩與以色列談判更難看，因為談判的前提是降溫，但現場訊號卻是擴大壓力。以色列說這是為了迫使 Hezbollah 撤離邊境、解除武裝；黎巴嫩和 Hezbollah 則說，以色列不停空襲又不撤軍，停火就只是名義。真正受害的還是南黎平民，因為撤離令聽起來像保護措施，但在長期衝突中也會變成新一輪流離失所。

加薩的停火也還卡在制度缺口。負責監督美國斡旋停火的人士要求聯合國安理會用一切可用手段施壓 Hamas 解除武裝，理由是任何暴力都可能讓協議瓦解。巴勒斯坦代表沒有直接回應解除武裝，而是強調各方都要遵守承諾；人道團體則提醒，和平計畫裡最容易做的援助准入，到現在都還沒有真正做好。這就是加薩停火的核心矛盾：安全安排沒有政治信任，人道安排又被安全安排卡住。

美洲今天有兩條選舉與街頭政治。哥倫比亞選民月底投票，選前無人機攻擊和武裝團體威脅讓安全成為主軸。選舉觀察組織列出三百八十六個市鎮有暴力脆弱性，智庫估計全國仍有大約兩萬七千名武裝成員。這其實是在問 Petro 的全面和平政策到底留下什麼：如果談判不能把武裝團體拉回政治程序，新政府就會在更分裂的安全環境裡上任。

玻利維亞則是街頭抗議逐步制度化成國家權力危機。總統 Rodrigo Paz 簽署新法，放寬軍方介入內部衝突的限制，背景是反政府抗議和道路封鎖已經持續接近一個月，政府也承認有示威者死亡。支持者會說，國家需要恢復交通、燃料和秩序；反對者會說，軍隊進入街頭只會讓經濟危機變成更深的合法性危機。這種情境最危險的地方，是雙方都覺得自己沒有退路。

北美供應鏈也出現新的摩擦。美國準備先和墨西哥談 USMCA 更新，焦點是經濟安全和關鍵工業品原產地規則，公告裡沒有加拿大。這個協定支撐接近一點六兆美元的三邊貿易，如果更新程序從三邊變成雙邊拼圖，企業最先感受到的不是關稅本身，而是不知道明年規則會長什麼樣子。

東亞今天的延續線索，是台灣軍售仍卡在川普政府決策裡。已經獲美國國會批准的對台一百四十億美元軍售還沒有順利推進，台北強調軍售是共同嚇阻，不應該被外部勢力當成交易籌碼；北京則反對任何對台軍售，把它說成干涉內政。這則沒有新的制度結果，但訊號很清楚：台灣正在面對的不是單純的軍事壓力，而是美中談判裡的不確定性。

菲律賓方面，Rodrigo Duterte 在國際刑事法院的反人類罪審判排定11月開庭，同一天菲律賓也有由主教和前 ICC 法官主導的新事實調查機制，要蒐集禁毒戰殺戮證據。這對 Marcos-Duterte 裂痕是一條長線壓力，因為國際司法、國內究責、Sara Duterte 的彈劾政治，會在接下來幾個月互相牽動。

非洲還有兩則戰爭外部化的新聞。Human Rights Watch 指控，一家與阿聯政府關係密切的阿布達比公司疑似招募並訓練哥倫比亞私營軍事承包人，部署到蘇丹支援 RSF。阿聯長期否認支援 RSF，但如果這條傭兵網路被更多證據坐實，蘇丹戰爭就不只是國內武裝衝突，而是外部資金、武器、傭兵和地方暴行疊在一起的複合戰爭。奈及利亞則說，和美軍合作的行動在數日內擊斃一百七十五名 ISIS 或 ISWAP 戰士。這是短期戰果，但還要看能不能獨立核實，以及能不能真的讓東北部社群安全下來。

大洋洲的焦點是毒品。太平洋今年前幾個月已經查獲約十七噸非法毒品，多數是 cocaine，Fiji 和澳洲聯邦警察因此推動跨國犯罪峰會。這個數字說明太平洋不再只是地圖上的轉運線，而可能成為庫存、洗錢、地方幫派和國際販運網路交會的地方。紐西蘭把十五點八億紐元國防方案放在海事安全，也可以放在這個背景下理解：島國的安全不是抽象的大國競爭，而是港口、航道、漁業、毒品和救災能力。

接著轉到科技。今天科技新聞的主線，是兩種基礎設施正在往主流走：一個是 AI agent 和本地模型，另一個是 stablecoin 支付。它們都不再只是早期使用者的玩具，所以風險也變得更現實。

先講 AI。Ollama v0.30.0 的預發版繼續往直接支援 llama.cpp 和 GGUF 格式推進，Apple Silicon 端也把 MLX 加速放進核心敘事。這聽起來像開發者工具的小版本，但它背後是本地 LLM 生態正在收斂：模型格式、推論後端、硬體加速，不可能永遠各走各的。對開發者來說，如果同一個 GGUF 模型可以更自然地在 Ollama 和 llama.cpp 之間流動，就少掉一份下載、少掉一層轉換，也少掉很多維運摩擦。不過，這仍然是預發版，真正值得看的不是 release note，而是不同模型大小、不同 Apple Silicon 記憶體配置下的實測速度。

LangChain 今天的幾個 release 則比較像 agent 生產環境的細修。Perplexity adapter 加了 use_responses_api 旗標，Fireworks adapter 修掉 content parts 裡非 wire keys 被送上 API 的問題。這種東西不會上頭條，但做過多模型 agent 的人會知道，很多事故不是模型突然變笨，而是訊息格式在框架、供應商、紀錄系統之間轉手時多了一個不該有的欄位。agent 產品要進企業，這些看似瑣碎的序列化、streaming、retry、profile data，會比 demo 裡的推理能力更常決定穩定度。

OpenClaw 也延續快版節奏，從 2026.5.26 的 Gateway、回覆、transcript 和 runtime cache 改進，推到 5月27日 的 alpha。這條線可以和更大的 AI 企業化放在一起看。Simon Willison 今天有一個很尖銳的觀察：OpenAI 和 Anthropic 可能已經在企業端找到產品市場契合，因為客戶真的開始照接近 API 標價付錢，不只是拿補貼試用。問題是，當一位工程師每月的 AI 使用成本進入幾百到幾千美元，AI 就不再是創新預算裡的實驗，而是會被財務部門逐項追問 ROI 的營運成本。

AI 帶來的另一面，是開源維護壓力。Daniel Stenberg 說，curl 團隊收到的可信安全回報速度，已經是 2024 年的四到五倍，是 2025 年的兩倍，平均每天超過一件。這很可能是 AI 輔助漏洞挖掘帶來的結果：回報更細、更像真的，但維護者一天還是只有二十四小時。這提醒我們，AI 安全工具如果只把更多工作丟給少數開源維護者，整個生態最後會把安全改善變成維護者健康問題。

加密和支付今天更像是「主流金融入口」的故事。Block 已經把 Cash App 的 stablecoin 功能分階段推給用戶，據報已經先開放給約四分之一用戶，目標是本週底讓近六千萬用戶都能使用。Mastercard 則取得紐約 BitLicense，讓它可以在更正式的監管框架下推 stablecoin 和 tokenized deposit 相關基礎設施。BIS 的 tokenization 專案也在說，跨境支付可能因 tokenization 變得更快、更安全。

這三件事放在一起，代表 crypto 的下一輪採用不一定是散戶交易，而可能是支付、結算和合規金融基礎設施。看多的人會說，這終於把鏈上美元帶進日常支付；看空的人會問，如果入口都由大型卡網、金融科技公司和監管許可把關，那 crypto 原本說的開放性還剩多少。比特幣短線跌破七萬五千美元附近、ETF 需求轉弱，也說明市場情緒沒有因為基礎設施好消息就單向樂觀。支付採用和資產價格，未必同步。

資安今天有一個 Android RAT 需要注意。BTMOB 被研究者描述成一種可以解鎖裝置、截圖、記錄按鍵、用 HTML injection 竊取憑證，甚至遠端控制手機的木馬。真正的問題不只是這個 RAT 有哪些功能，而是 RAT-as-a-Service 加 no-code builder 讓攻擊門檻下降。當製作惡意 app 的流程越來越像合法 app builder，金融詐騙的規模化就會比單一漏洞更難處理。

太空方面，SpaceX 完成 Starship V3 的首次測試飛行，這是 NASA 後續 Artemis 月球任務押注的關鍵系統之一。這則在過去幾天已經講過，所以今天只補一句：成功測試提高工程信心，但載人登月需要的是重複可靠、軌道補給、登月器整合和安全驗證，不是單次漂亮發射。把這點記住，就比較不會被每一次壯觀畫面帶著走。

最後，AI 生醫今天不是大新聞，但有一個方向值得留意。近期論文開始比較專用藥物資產發現平台和 frontier LLM 在小眾腫瘤、免疫標的上的表現，結論偏向 curated AI 更可靠。這其實很合理：生醫 AI 的護城河不一定是最大模型，而是高品質資料、標註、專家流程和可稽核來源。也就是說，越進入專業領域，模型能力越像入場券，資料和流程才是比賽本身。

今天先到這裡。總結一下，世界新聞的關鍵是外溢：疫情外溢、戰爭外溢、毒品網路外溢、軍售不確定性外溢。科技新聞的關鍵是落地：AI 開始落進企業成本，stablecoin 開始落進支付入口，本地模型開始落進實際硬體與格式選擇。這兩條線看起來很不同，但共同點是，一旦事情進入真實基礎設施，就不再只看敘事，而要看治理、成本和可靠性。明天見。

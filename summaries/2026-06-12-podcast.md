第100集，6月12日。

今天這集，先看一個很不穩定的外交時刻：美國和伊朗一邊互相開火，一邊又傳出接近協議。這不是和平已經到來，而是雙方都在試著把軍事壓力翻譯成談判籌碼。第二條主線，是台灣首次把 HIMARS 對著西岸戰略水域實彈演訓，台海訊號從軍購清單，變成更可見的火力展示。後半段轉到科技，今天的關鍵字是 agent。當 AI agent 可以交易、付款、操作工具，安全問題就不再只是模型會不會胡說，而是它能不能替你花錢、替你承擔風險。

先從時事開始。

美國和伊朗，是今天最需要小心解讀的新聞。Trump 說，美國和伊朗接近簽署和平協議，甚至因此取消部分原本可能執行的飛彈攻擊。這句話聽起來像重大突破，但它其實還不是停火完成。伊朗媒體對美方說法沒有完全背書，談判條件也還卡在幾個很硬的問題：制裁要怎麼鬆綁，伊朗資產要不要解凍，核計畫要怎麼被核查，還有荷姆茲海峽的通行安全誰來保證。

這裡的重點，是為什麼雙方現在都需要一個可以下台階的說法。美國剛經歷對伊朗城市和軍事設施的空襲，伊朗也用飛彈和無人機打向 Bahrain、Kuwait、Jordan 等地的美軍相關基地。雙方都已經讓衝突升級，所以如果突然退回談判桌，就需要各自對內說明：這不是退讓，而是策略成功。

美國的敘事，是用軍事壓力逼 Tehran 回到條件更嚴格的協議。伊朗的敘事，則會避免讓外界看成自己在空襲後被迫簽字。這個差異很重要。因為一份真正能穩住局勢的協議，不只是兩邊願意停火，還要讓兩邊都能在國內政治裡活下來。

真正麻煩的，是 Lebanon 和 Gulf bases 這些外部戰線。美國通常會想把談判範圍切窄，聚焦核計畫、制裁和海峽通行。伊朗則會把 Hezbollah、Lebanon 戰線和地區安全一併拉進來。Israel 又不一定接受任何會限制它在 Lebanon 行動的安排。所以你會看到一個很矛盾的局面：華府說協議快了，Tehran 沒有完全承認，Israel 仍在 Lebanon 維持軍事行動，Hezbollah 也不會因為美伊談判就自動接受撤退條件。

這就是今天美伊消息的讀法：它是一個降溫機會，但不是和平事實。你可以把它想成雙方在懸崖邊同時伸手找欄杆。抓到了，可能往回走；抓不到，下一次軍事行動就會被包裝成「對方破壞談判」。

同一條能源與安全線，正在影響歐洲。European Central Bank 把 deposit rate 從兩個百分點升到二點二五個百分點，這是 2023 年後第一次升息。原因是 Iran war 推高能源成本，May eurozone inflation 到三點二個百分點，高於 ECB 的兩個百分點目標。

這個決定的政治味很濃。央行怕重演二零二二年那種能源通膨追不上、政策慢半拍的場景，所以先選擇防通膨。但如果美伊很快降溫，歐洲就可能在景氣變弱時承擔多一次升息的成本。美國國內也被同一件事拉住。May CPI 年增四點二個百分點，core CPI 年增二點九個百分點。戰爭如果推高能源，貨幣政策就不能只看國內需求。

接著看台灣。台灣首次在西岸附近，以美製 HIMARS 對面向中國的水域實射三十二枚火箭。這個動作有戰術意義，也有政治意義。戰術上，HIMARS 是機動精準火力，台灣把它放進反登陸演訓，是要讓對岸相信，登陸船團、集結區或後續補給點會被遠程火力威脅。政治上，這發生在美國對台一百四十億美元軍售包裹仍有延宕的背景下，所以它同時是給北京看，也是給華府看。

台北的語言會是防衛和嚇阻。北京的語言會是挑釁和「以武謀獨」。這兩套語言幾乎沒有交集。台灣如果不展示能力，嚇阻會被質疑；台灣如果展示能力，北京又會把它納入升高軍事壓力的理由。這就是台海訊號最難的地方：同一個行動，在三個首都裡會被翻譯成三種意思。台北看成保命，北京看成紅線，Washington 可能看成談判籌碼和可信度測試。

East Asia 另一條線，是 Xi Jinping 罕見訪問 North Korea。北京沒有把 denuclearization 放在最前面，這件事本身就值得 Seoul 和 Tokyo 仔細解讀。Kim Jong Un 這幾年靠近 Russia，支援 Moscow 的戰爭，也得到更大的外交空間。對 China 來說，如果 North Korea 完全倒向 Russia，北京在半島上的槓桿會變小；所以 Xi 訪問 Pyongyang，是重新確認影響力。這不是說中國正式承認北韓核武，而是公開壓力降低，本身就是 Kim 的外交收穫。

South China Sea 也有新事。中國宣布禁止 Philippines defense secretary Gilberto Teodoro 和家人入境 China、Hong Kong，原因是他長期批評 Beijing 的南海主張。這件事把海上對峙延伸到個人制裁。Manila 會更有理由把二零一六年仲裁結果和盟友安全合作放在前台；Beijing 則是在說，對中國主權主張的高調挑戰會有外交成本。

南亞方面，Pakistan-administered Kashmir 的直升機墜毀今天更具體了。二十二名軍人死亡，官方初判是 technical fault，沒有說它和 Joint Awami Action Committee 的抗議有直接關聯。但這起事故發生在 Muzaffarabad 周邊安全部署升高時，機上人員也是前往支援安全任務，所以即使不是抗議造成，社會感受也很難把兩件事完全切開。

JAAC 的背景，是民生和政治代表權。它提出過三十八點要求，包括補貼小麥、電力，以及十二個保留給 Kashmiri refugees 的議席問題。政府說三十六點已經接受，剩下的還要談；JAAC 則堅持繼續動員。當一個地方同時有自治代表性爭議、民生壓力、禁令、逮捕和軍警部署，任何事故都會增加不信任。

Middle East 另一條線，是 West Bank settler sanctions。Australia、UK、Canada、France、Norway、New Zealand 對涉入 settler violence 和 settlement expansion 的個人與實體追加制裁，France 還針對 Israeli finance minister Bezalel Smotrich 採取限制。Israel 會批評這是在獎勵反以色列力量；制裁國則會說，West Bank 的定居點暴力和 outposts 擴張，已經不能只靠口頭譴責處理。

這件事的變化，是 Gaza、Lebanon、West Bank 不再被盟友完全分開看。過去很多政府會把軍事戰線和定居點政策放在不同抽屜；現在它們開始被放到同一張外交桌上。這對 Israel 的壓力不一定立刻變成政策改變，但會讓它和歐洲、英聯邦盟友的關係更難管理。

Africa 今天最重要的是 Congo Ebola。最新數字是五百五十例 confirmed cases，一百零一人死亡，十九人康復，主要集中在 Ituri，也擴及 North Kivu、South Kivu 和 Uganda。更麻煩的是，這次是 Bundibugyo strain，沒有成熟批准疫苗或治療。接觸者追蹤 coverage 雖然改善，但仍大約只有六十四個百分點。

這裡的問題不是只有病毒，而是衝突環境。近百萬人流離失所，礦工流動，部分地區有武裝活動，醫療人員還面臨攻擊和不信任。公衛系統最怕的，就是病毒傳播路線和戰爭、逃難、工作路線重疊。Ethiopia election 也有類似治理壓力：超過五千萬登記選民參與投票，但 Oromia 和 Amhara 的安全事件讓一百四十三個 polling stations 沒有重新開放，Tigray 也連續第二次沒有參與。

美洲除了美國通膨，還有 Brazil 和 Colombia。Brazil 報告說，May Amazon deforestation 比去年同月下降六十一點四個百分點，去年八月到今年五月也年減三十七點五個百分點。Lula 政府用這些數字反擊 Trump administration 對巴西商品擬課二十五個百分點 tariff 的環境理由。Colombia 6月21日決選，則會決定 Amazon、石油和 Indigenous rights 的路線。Iván Cepeda 延續 Petro 的環境路線，支持停止新油氣探勘和提高原住民參與；Abelardo de la Espriella 偏向 fracking、化石燃料和更強硬治安。

Oceania 方面，Australia 和 Solomon Islands 正在重設安全關係。Matthew Wale 上任後訪問 Australia，雙方同意推進 comprehensive strategic treaty 和 joint policing initiative，Australia 也承諾約三千五百萬澳元支援 cyclone recovery 和能源衝擊。Canberra 想成為 Pacific security partner of choice；Solomon Islands 則想在 Australia、China、New Zealand 和 United States 之間保留主權空間。這不是單純倒向哪一邊，而是小國在大國競逐下怎麼爭取選擇權。

最後一條氣候線。El Niño 已經在 Pacific 形成，NOAA experts 說二零二六到二零二七這一輪，有六十三個百分點機率接近一九五零年以來最強等級；WMO 先前也把六月到八月形成機率估到八十個百分點。這會影響 Indian monsoon、Australia drought risk、California winter rainfall，還有 South Africa 和 Southeast Asia 的糧食供應。

比較短的延續更新一起帶過。Ukraine 方面，Zelenskyy 在 London 對 Starmer、Macron、Merz 要求擴大防空與 deep-strike 支援；Israel-Lebanon 戰線仍是美伊談判的外部破口；Philippines 和 Vietnam 的 South China Sea 合作還在延續；USMCA 二零二六年檢視期逼近，北美供應鏈仍在等三國能不能給出更長期確定性。

從時事看下來，今天的共同問題是：國家怎麼在危機中畫邊界。美伊在畫戰爭和談判的邊界，台灣在畫防衛展示和政治升級的邊界，Ebola 在畫旅行自由和公共衛生的邊界，Solomon Islands 在畫主權和安全依賴的邊界。接下來轉到科技，你會發現科技新聞也在問同一件事，只是主角從國家變成 AI agent、wallet 和 software framework。

科技第一條，是 Coinbase 推 AI agent accounts。這件事很重要，因為它讓 agent 不只是回答問題，也不只是呼叫 API，而是可以建立帳戶、交易、付款。你可以想像一個 agent 幫你買雲端算力、幫遊戲角色處理鏈上資產、幫自動化服務支付小額費用。這些都很有效率，但每一個便利都會立刻變成風險問題：誰授權？限額多少？錯付能不能追回？被 prompt injection 誘導交易怎麼辦？

過去我們談 agent safety，常常停在模型會不會亂下指令。現在問題更硬：如果亂下指令會花錢，甚至會交易資產，那就不是一個聊天機器人的小錯，而是金融風控。支持者會說，agent economy 需要原生付款層；批評者會說，金融世界最怕的不是沒有自動化，而是自動化太快、責任太慢。

第二條，Canton Network 開發商融資三點五五億美元，要把 Wall Street assets 帶到 onchain rails。這和 Coinbase agent accounts 放在一起看，會看到一個更大的趨勢：加密敘事正在從「買幣」轉成「金融基礎設施重寫」。大型銀行不再只問誰會成為 stablecoin winner，而是問 tokenized cash、private markets、settlement 和 compliance 怎麼接到公鏈或類公鏈基礎設施。

看多的人會說，這代表鏈上金融終於進入機構主線。看空的人會問，法律所有權、託管、二級流動性、KYC 和跨境合規，是否真的會因為上鏈就變簡單。事實上，它不會自動變簡單。上鏈只是把記錄和轉移方式改了，真正難的是法律和市場結構要跟上。Citi 的 tokenized share offering 也是同一方向：如果估值不透明、退出管道不足，投資人保護沒有變好，那它只是把舊市場換一層新介面。

第三條，是 Anthropic 調整 Claude 對 frontier LLM research 的 safeguard 顯示方式。研究者擔心的不是安全機制存在，而是安全機制如果不可見，就可能讓實驗結果無法重現。你以為模型能力不行，結果其實是 safeguard 在背景中改寫或阻擋；你以為某個 benchmark 有意義，結果不同研究者碰到的限制不一樣。

這件事對 AI safety 很關鍵。安全限制需要存在，但它也需要可觀測、可解釋、可審計。否則安全就會和科學可重現性衝突。Anthropic 把限制變得更可見，是在承認一件事：frontier model 研究不能靠黑箱規則維持信任。

LLM 商業面，Gary Marcus 說 OpenAI 可能考慮大幅降價。他把這解讀為需求和競爭壓力訊號。這不一定代表 OpenAI 弱，也可能只是基礎模型 API 正在走向雲服務價格戰。但它會衝擊一個長期問題：frontier lab 的估值，是建立在模型能力稀缺和高毛利上。如果價格快速下探，開發者受益，使用量上升，但訓練成本、推論補貼和投資回收就會被重新估算。

Datasette 1.0a33 和 datasette-agent 0.2a0，則是 agent 工程裡比較務實的更新。Datasette 把 `_extra=` API pattern 擴到 queries 和 rows，讓工具可以按需取回更多延伸資訊。datasette-agent 則讓工具在執行中向使用者提問。這聽起來很小，但其實是人機協作的核心：agent 不應該在資訊不足時硬做決定，它應該能暫停、問問題、拿到確認，再繼續。

LangChain Core 1.4.6 修正 streamed tool calls 正規化問題，model profiles 0.0.6 也增加 streaming tool call chunks 測試。這些 release 不會上主流頭條，但對真正部署 agent 的團隊很重要。因為 agent 的可靠性常常死在邊界格式：串流時工具呼叫片段不穩、不同模型輸出欄位不一致、tracing metadata 不足，最後就會變成很難重試、很難回放、很難找責任。

PyTorch 2.13.0 release candidate 也有更新，主要是 release wheel build 流程修補。大型框架的 wheel 產線穩定，會影響下游 library、企業部署和研究環境重現。Hugging Face 的 PyTorch fused MLP profiling 文章也提醒我們，推論成本不只靠模型大小，也靠 kernel fusion、記憶體路徑和硬體利用率。

Consumer tech 今天還是 Apple Siri AI 的延續。WWDC 之後，Apple 的故事變成「終於快來了，但還沒完全來」。新 Siri AI 有多模態、個人上下文、跨裝置和 Private Cloud Compute，但很多能力仍然要等 public beta 和地區開放。Apple 的優勢，是它可以把 AI 放進 iPhone、Mac、Watch、Vision Pro 這些日常入口；它的風險，是如果 rollout 太慢，使用者會覺得又一次承諾先行。

Cybersecurity 今天也被 agent 金融化拉進來。當 Coinbase、Mastercard 和各種 wallet infrastructure 讓 agent 能付款，安全問題不只是帳密被偷。它會變成 agent key 怎麼保管，wallet policy 怎麼設限，異常交易怎麼偵測，使用者怎麼撤銷授權，平台怎麼賠付。Prompt injection、釣魚、惡意 tool output，最後都可能變成資產操作。

美國眾院擬建立 crypto-theft task force，也在回應同一個現實：加密竊盜不是單一交易所事件，而是跨國犯罪、洗錢、社工和基礎設施漏洞組成的網路。支持者會說，跨機關任務小組能提高追回和威懾；隱私倡議者則會擔心，執法擴權如果缺乏透明度，會壓縮正當隱私工具。

今天的科技段落共同指向一件事：agent 正在從文字介面走向金融介面、資料介面和操作介面。當它只會回答問題，你最怕的是錯誤資訊；當它可以付款、交易、查資料庫、呼叫工具，你最怕的是錯誤行動。接下來幾年，真正的競爭不會只在誰的模型更聰明，也會在誰能把授權、審計、回滾和責任做得更清楚。

收尾來看，今天時事和科技其實都在講同一個字：控制。國家想控制戰爭升級，央行想控制通膨，台灣想控制嚇阻訊號，公衛系統想控制病毒路徑；科技公司則想控制越來越能行動的 agent。控制不是把所有風險消掉，而是讓風險出現時，還有人知道發生了什麼、誰能按下暫停、誰需要負責。

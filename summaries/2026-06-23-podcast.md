EP110。今天是6月23日。

早安。今天這集，我想先從三個壓力點開始：哥倫比亞總統決選揭曉，右派以很小的差距拿下政權；美伊談判看似找到六十天窗口，但黎巴嫩戰場還沒有真正被納入同一張桌子；東剛果 Ebola 確診突破一千例，衝突區的公共衛生系統已經被推到邊緣。科技線則看兩件事：agent 進企業以後，真正缺的是治理；以及 crypto 從價格故事，越來越變成監管、公司治理和鏈上對抗的故事。

先從時事開始。

今天第一個新事，是哥倫比亞總統決選終於開出結果。右派律師、富豪 Abelardo de la Espriella，以百分之四十九點六六的得票率，擊敗左派參議員 Iván Cepeda 的百分之四十八點七。差距大約二十五萬票，聽起來不是照片判讀等級，但對一個高度撕裂的國家來說，已經足夠讓輸的一方質疑制度，也足夠讓贏的一方宣稱取得授權。

這件事之所以重要，不只是因為哥倫比亞換黨。它是今天時事線最明確的類別變化事件：選舉結果公布，而且政權方向有明顯轉彎。過去幾年，Gustavo Petro 政府把「全面和平」當成核心路線，試著用談判處理 guerrilla、毒品組織、地方武裝和國家治理失靈。但很多選民看到的是治安沒有真正改善，綁架、勒索、地方控制依然存在，所以 de la Espriella 的強硬路線有了空間。

支持 de la Espriella 的人會說，哥倫比亞不能永遠把武裝團體當成談判對象，國家必須恢復威懾，否則和平只是讓對方重整隊伍。支持 Cepeda 和 Petro 路線的人則會說，哥倫比亞已經太熟悉軍事掃蕩的代價：鄉村社群被夾在軍隊、游擊隊和毒品網絡中間，短期治安數字改善，長期卻沒有把地方經濟和國家服務補上。

這場選舉還會外溢到拉美政治。de la Espriella 主張更靠近美國，也更支持強硬反毒行動；如果他真的結束部分和平談判，哥倫比亞可能會重新進入一種更軍事化的治安周期。反過來說，他也必須面對很現實的問題：如果只是把強硬口號變成空中掃毒、軍警進駐，卻沒有司法、土地、財政和地方治理，那麼暴力很可能只是換一個名字回來。今天的哥倫比亞不是在選「和平」或「安全」其中一個，而是在問：當和平承諾失去耐心時，社會會不會重新把安全交給強人政治。

第二個新事在中東。美國和伊朗在瑞士談判後，都釋出「有進展」的訊號。美方說這為最終協議打下很好的基礎，伊朗同意讓 IAEA inspectors 回來，美國則提供到8月21日的伊朗油品制裁暫時豁免。表面上，這是一個外交降溫；實際上，最大的變數還是在黎巴嫩。

因為這輪安排裡，有一個美伊和黎巴嫩之間的 deconfliction cell，目標是降低誤判，避免 Lebanon ceasefire 破裂。以色列總理 Netanyahu 已經說得很清楚：就算有這個機制，IDF 在南黎巴嫩仍然保留完整行動自由。這句話等於把談判的弱點說破了。華府和德黑蘭可以談通訊管道，可以談荷姆茲海峽，可以談制裁豁免；可是如果以色列和 Hezbollah 不是機制中的核心約束對象，前線規則就不一定會照著外交文件走。

伊朗的立場也很明確。它把制裁、核查、黎巴嫩撤軍和海峽航行安全放進同一組政治交換裡。美國想把每一題拆開處理，因為拆開才有管理空間；伊朗想把它們綁在一起，因為綁在一起才有槓桿。這就是中東談判最難的地方：每一個議題都有自己的當事人，但每一個當事人又都試著把其他議題當籌碼。

所以今天中東的重點不是「和平快到了」。比較準確的說法是，戰爭各方都找到一個可以暫時不升高的理由。這很重要，因為不升高本身就能救命；但它還不是穩定，因為黎巴嫩南部、以色列北部、伊朗核查和荷姆茲航道，任一條線都可能重新拉動整個局面。

第三個新事是東剛果 Ebola。AP 報導，Ituri 省的 outbreak 已經達到一千零三例 confirmed cases，二百五十四人死亡，一百人康復，至少三百六十五人住院或隔離。最麻煩的是，contact tracing coverage 只有百分之五十五，而且這次是 Bundibugyo strain，目前沒有可用疫苗或特效治療。

這裡的風險不是單純病例數字變大，而是病例出現在衝突和 displacement 的地理裡。UNHCR 警告，有大約兩百萬 displaced persons 住在 Ebola-risk zones。你可以想像，一個擁擠營區裡，水、廁所、隔離空間、交通、醫護防護都不足，病毒不需要太多時間就能穿過家庭、鄰里和照護者。更糟的是，當地還有武裝衝突，很多社區不是醫療隊想進就能進。

公衛危機最怕兩件事一起發生：第一，病原體難處理；第二，制度沒有信任。Ebola 需要快速隔離、追蹤接觸者、改變喪葬習俗，這些都很容易和地方社群的恐懼衝突。如果居民覺得政府和外來醫療人員只是帶來禁令，卻沒有帶來安全和照顧，病例就會被藏起來。這就是為什麼今天的數字很重要：一千例不是只代表疫情變大，它代表控制窗口正在縮小。

這條新聞也提醒我們，全球公共衛生不能只在疫情跨境以後才被看見。當一個 outbreak 還在地圖上的偏遠省分，國際注意力往往很低；等到病例進入大城市、難民營或跨境通道，處理成本就會成倍增加。Ebola 的教訓一直很清楚：早期信任、地方照護和接觸者追蹤，比後期封鎖更便宜，也更有效。

接著看歐洲。Keir Starmer 宣布辭任英國首相後，原訂7月22日的 EU-UK summit 被迫延期。這場峰會原本要談食品標準、emissions trading、youth mobility，也會觸及英國是否更深入參與歐洲防務融資。歐盟領袖對 Starmer 的評價普遍正面，Zelenskyy 也肯定他對烏克蘭的支持，但政治現實是，英國對歐關係又要等新首相重新確認授權。

Starmer 時代的意義，是把 Brexit 之後的英歐關係從純粹敵意，拉回到安全合作和實務修補。可是這種修補一直有天花板。英國不想重新打開自由流動和主權讓渡的內戰，歐盟也不想讓英國只拿好處、不接受規則。現在 Starmer 下台，問題就變成：英國接班人會把靠近歐盟視為必要現實，還是把它當成政治負擔。

同一天，德國計畫持有 KNDS 百分之四十股權，讓這家 Leopard 和 Leclerc 坦克製造商成為更明確的國家戰略資產。KNDS 有超過一萬一千名員工，年營收約四十四億歐元。這不是一筆普通產業投資，而是歐洲在俄烏戰爭和美國政策不確定下，重新定義軍工產能。過去歐洲常把防務外包給美國安全傘，現在越來越多國家開始問：如果美國的承諾變得不穩，歐洲自己的產線能不能撐住。

烏克蘭線上，基輔加強打擊克里米亞的補給與交通，目標包括 Novorossiya highway、Kerch Bridge 相關物流、油庫和渡輪。克里米亞出現民用燃油銷售暫停、加油站排隊，甚至兒童夏令營關閉。烏克蘭把這稱為 long-range sanctions，也就是用遠程打擊提高佔領成本。俄羅斯則會說這是攻擊民生。這場戰爭正在變成一種後方壓力戰：不是只看前線村莊推進幾公里，而是看誰能讓對方的補給、旅遊、能源和日常秩序都變得更貴。

東亞和台灣線上，台灣6月22日開始五天 Immediate Combat Readiness Exercises。AP 說，在桃園可以看到裝甲車和坦克在市區與高速公路巡邏。台灣國防部同時通報，週日到週一上午有二十三架共機、七艘軍艦和五艘中國公務船靠近台灣。這不是年度大拜拜式軍演，而是把灰色地帶突然升高時，部隊能不能快速部署，直接搬到城市和交通節點裡測。

這件事要和一百四十億美元美國軍售案一起看。台灣駐美代表俞大㵢先前說，台灣不會等美國騎兵，但仍需要美國武器補足防衛。這句話的矛盾，正是台灣安全的現實：台北必須展現自己能動員、能訓練、能承受壓力，但很多關鍵能力仍被華府交付節奏牽動。北京看見的是挑釁，台北看見的是嚇阻，華府看見的則是對中談判和安全承諾之間的槓桿。

幾條延續快速帶過。蘇丹無人機戰今年前五個月已殺害超過一千名平民，這條線昨天講過，今天要補的是 UN 對 accountability vacuum 的壓力繼續升高。澳洲破獲史上最大 cocaine seizure，在悉尼西北查獲二點七噸、街價約八點一六億澳元，這讓太平洋與澳洲東岸的跨境犯罪網絡重新被看見。澳洲把部分非公民送往 Nauru 的政策仍在發酵，問題不只是遣返，而是三十年簽證、家庭分離和長期流放。印度 Ladakh 團體今天 shutdown，抗議中央政府沒有把自治會議承諾寫進正式紀錄，這是邊境治理和地方信任問題，不只是地方政治新聞。

如果把今天時事線收束成一句話，就是：很多制度都在被迫回答「誰有最後決定權」。哥倫比亞選民把安全方向交給新右派，英國要重新決定對歐路線，台灣要在美國交付節奏之外證明自我防衛，剛果則在問，當國家控制不了衝突區時，公共衛生到底怎麼落地。

好，轉到科技。

今天科技第一個新事，是 go-ethereum 釋出 v1.17.4。這是一個 maintenance release，不是 flashy product launch，但它有兩個值得注意的點。第一，它把 standalone Clef signer 移出 bundled release，放到獨立 repo。第二，它延續 Amsterdam hardfork 的實作準備。

為什麼這種看起來很工程的小更新值得講？因為 Ethereum 的基礎設施不是一個網站更新就好，它是節點營運者、交易所、託管商、錢包服務、內部安全流程一起跑的系統。Signer 工具拆出去，對一般使用者可能沒有感覺，但對營運團隊來說，就代表部署、文件、監控和應急流程都要檢查一次。區塊鏈最常被外界看見的是價格和敘事，真正決定它能不能撐住下一輪使用量的，常常是這些無聊但必要的維護版。

第二個新事，是英國央行調整 sterling stablecoin 的監管設計。它放棄較具爭議的個人持有上限，改用四百億英鎊的 systemic stablecoin issuance guardrail，目標時間仍指向2027年。這個轉向很有意思，因為它承認支付型穩定幣如果真的要用，不能一開始就把個人使用卡得太死；但央行仍然要防止總量大到威脅銀行存款和金融穩定。

看多方會說，這是讓合規穩定幣走向日常支付的必要一步。你不能一邊說要創新，一邊讓使用者每一步都碰到 cap。看空方會說，去掉個人上限不代表風險消失，只是把風險從個人層級搬到系統總量層級。如果大量資金從銀行存款轉向穩定幣，商業銀行的資金來源、支付清算和危機時刻的擠兌路徑都會改變。這不是 crypto 圈自己的問題，而是貨幣體系的問題。

第三個新事，還是在 tokenization。Bernstein 的數字顯示，tokenized real-world assets market cap 上升四成，超過五百一十億美元，其中 equity tokenization 成長一百三十個百分點。這代表「把現實資產搬上鏈」已經不只是小型實驗，而是機構金融開始搶產品定義權。

但這裡要小心。RWA 的支持者很容易把它講成效率故事：結算更快、門檻更低、跨境更方便。這些都可能成立。問題是，資產上鏈以後，法律所有權在哪裡？投資人如果在不同司法管轄區交易，誰負責揭露、託管、破產隔離和爭議處理？如果 token 有流動性，但底層資產沒有流動性，危機時刻會不會只是把舊金融的錯配包成新介面？所以 RWA 的真正競爭，不是誰先發 token，而是誰能把法律、託管和二級市場一起做成可信基礎設施。

接著是 AI agent。Cloudflare temporary deploy 被很多人拿來看作 agent 友善的 ephemeral workspace。開發者可以用一個 temporary deploy，把 Workers project 快速部署到一個短生命週期環境，大約保留六十分鐘。這對 agent 很有吸引力，因為 agent 最擅長的就是快速試作、跑通、丟掉，再下一輪。

可是這也正好暴露 agent tooling 的治理問題。短生命週期環境可以讓試作變便宜，也可以讓錯誤更快地被丟到外部網路。誰能建立這個 temporary project？它可以連到哪些 secrets？它產生的 endpoint 有沒有成本上限？它到期後 logs 和資料怎麼處理？agent infrastructure 的下一步，不只是讓 agent 更容易 deploy，而是讓 deploy 的邊界、權限和清理都能被預設管好。

PyTorch 今天也有兩個值得注意的小更新。一個是 CUTLASS row-wise scaled fp8 matmul 增加 fp32 output 支援；另一個是 MPS backend 遇到 complex dtype 的 avg_pool 時，改成和 CPU、CUDA 更一致的 NotImplemented error。這些不是會登上發表會的大功能，但對做低精度加速、跨平台模型開發的人很實際。尤其 Apple Silicon 上的 ML stack，很多團隊在 prototype 階段跑得很順，到了不同 dtype、不同 backend、不同 operator 邊界才發現行為不一致。今天這種修補，就是把「偶然爆掉」改成「明確拒絕」，工程上很重要。

AI 另一條是 PP-OCRv6 登上 Hugging Face，主打五十種語言 OCR，模型尺寸從一百五十萬到三千四百五十萬 parameters。文件 AI 這一年很熱，但 OCR 其實是很多企業流程的第一道門。發票、合約、醫療單、掃描表格，模型如果連字都讀不穩，後面的 reasoning 再漂亮也沒有用。小模型的優勢是能跑在邊緣和低成本設備上；風險是跨語言、低解析度、手寫、表格結構，仍然需要透明 benchmark。

生技線上，AbbVie 以大約二十一億美元收購 Gilgamesh 的 depression drug candidate Bretisilocin program。這筆交易把 psychedelic-derived therapy 又推回大型藥廠視野。它不是典型 AI drug discovery 新聞，但很能說明資本仍在尋找精神醫學的新機制。風險也很清楚：憂鬱症藥物的臨床終點、給藥場景、監管要求和支付端接受度，都比新聞標題慢很多。

幾條科技延續。OpenClaw 釋出 beta notes，提到短對話 fast mode、model routing 和 bounded fallback，這是 agent runtime 高頻迭代的一部分；但高頻也代表回歸測試和可觀測性要跟上。Solmate 最大股東控告 board self-dealing，稱 SLMT 股東被稀釋約二成，這提醒 crypto treasury 熱潮正在把 token 敘事搬進上市公司治理。Spot bitcoin ETF 連續第六週淨流出，上週淨流出二點二七億美元，分析師一邊說賣壓可能接近尾聲，另一邊也說機構需求還沒有真正恢復。

今天科技線的共同主題，是「基礎設施化」。Ethereum client 更新是基礎設施，stablecoin guardrail 是金融基礎設施，RWA 是資產市場基礎設施，Cloudflare temporary deploy 是 agent 基礎設施，PyTorch 是模型執行基礎設施。基礎設施的特徵是，它成功時沒有人注意，失敗時所有人都會被拖下水。

所以我今天不太想用「突破」來形容科技新聞。更準確的是，技術正在進入制度層。agent 不只是 demo，要進企業權限系統；stablecoin 不只是交易對，要進支付和央行框架；RWA 不只是鏈上資產，要接法律所有權；OCR 不只是模型分數，要接企業文件流程。這些事沒有單一華麗瞬間，但它們會決定下一階段誰能真的被採用。

最後，把時事和科技放在一起看，今天有一條共同線：工具越強，制度越不能缺席。哥倫比亞的新政府可以選擇強硬工具，但如果制度不修，暴力會換形式回來。台灣可以展示快速部署工具，但如果交付、訓練和彈藥沒有穩定，嚇阻就只是訊號。AI agent 可以快速部署，stablecoin 可以快速轉帳，RWA 可以快速結算，但如果權限、法律和問責沒有跟上，速度本身就會變成風險。

今天就到這裡。6月23日，時事線看哥倫比亞政權右轉、美伊談判的六十天窗口、東剛果 Ebola 破千例，以及台灣即戰力演練；科技線看 Ethereum client 維護、英國穩定幣監管、RWA 成長、agent temporary deploy 和底層 ML runtime。明天我們繼續。

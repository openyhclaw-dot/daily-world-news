今天是5月25日，歡迎收聽第八十二集每日世界新聞。

今天的節奏很清楚：世界線上，戰爭沒有結束，但好幾個戰場都進入「看起來像降溫、其實還很危險」的階段。科技線上，AI 不只是模型能力往前跑，而是開始被宗教、供應鏈、太空運輸和開發者工作流一起重新定義。

我們先從時事開始。

今天最值得放在前面的新事，是俄羅斯對基輔的攻擊。烏克蘭總統澤倫斯基說，俄羅斯在新一輪大規模無人機與飛彈攻擊中使用了 Oreshnik 高超音速彈道飛彈，至少造成兩人死亡。這是四年戰爭中，這種武器第三次被使用。

這件事不能只看成又一輪空襲。Oreshnik 的政治訊號很強，因為俄羅斯不是只在消耗烏克蘭防空，也是在提醒歐洲：就算制裁一輪接一輪，莫斯科仍然想展示它有升級工具。對烏克蘭來說，這會把問題拉回兩個最現實的點：防空彈藥夠不夠，以及歐洲能不能把承諾變成穩定交付。

歐盟第20輪制裁已經把能源收入、軍工供應鏈、金融服務，甚至加密資產通道都納入範圍。但制裁的效果通常是中長期的，基輔今晚面對的卻是即時的飛彈和無人機。這就是烏克蘭戰爭現在最難的地方：西方政策工具的時間尺度，跟戰場上的耗損速度，常常不同步。

第二個新事在非洲，剛果的 Ebola 疫情正在變得更複雜。WHO 已經把剛果和烏干達的 Bundibugyo Ebola 疫情列為國際關注公共衛生緊急事件。AP 今天報導，剛果東部一間治療 Ebola 病患的醫院遭到憤怒民眾闖入，醫護人員被迫撤離病患，現場還出現槍聲。這已經是一週內第三起針對相關醫療設施的攻擊。

數字本身很刺眼。剛果當局說，東部疑似病例已經超過九百例；WHO 的疫情說明也提到，病例正在從 Ituri 向 North Kivu、South Kivu 擴散。更麻煩的是，這次是 Bundibugyo virus disease，目前沒有已核准疫苗，也沒有特效治療。這讓防疫回到很傳統、也很辛苦的幾件事：早期辨識、隔離、接觸者追蹤、社區溝通，以及安全的治療中心。

但你會注意到，這次的問題不是單純醫療資源不夠。治療中心被攻擊，代表防疫措施已經碰到社區信任崩塌。很多疫情到這一步，國際機構會說要增加 PPE、藥品、實驗室與快速反應隊；地方社群感受到的卻可能是外人把親屬隔離、限制喪葬、甚至帶走遺體。兩邊如果沒有共同語言，防疫本身就會被當成敵人。

第三個新事是賽普勒斯國會選舉結果。DISY 保住第一大黨，拿下大約百分之二十七點二的票，保有五十六席中的十七席；但真正值得注意的是 ELAM。這個過去與希臘金色黎明有連結的極右政黨，得票從二零二一年的百分之六點八升到百分之十點九，席次從四席翻倍到八席，成為第三大勢力。

這次國會選舉不會立刻讓政府垮台，可是它會改變議題的重心。移民、歐盟、土耳其、國家安全，接下來會更常被放進右派敘事裡。這也是歐洲最近幾年很熟悉的畫面：傳統大黨沒有完全崩盤，但它們旁邊的激進右派開始穩定長大，並逼主流政黨改變語氣。

接著看美洲。美國起訴古巴前領導人 Raúl Castro，指控他和一九九六年擊落 Brothers to the Rescue 飛機案有關。古巴總統 Miguel Díaz-Canel 批評這是政治表演。這件事法律上很老，政治上卻很新，因為它發生在川普政府對古巴壓力升高的時候。華府把它包成司法追責，哈瓦那看到的是政權更替壓力。兩邊都知道，Raúl Castro 不太可能真的走進美國法庭，但起訴本身會成為新的外交武器。

玻利維亞則是治理危機正在加深。Rodrigo Paz 上任不到半年，首都 La Paz 就被抗議和封鎖拖進圍城狀態，Evo Morales 的支持者也加入要求政府下台。這不是一場單一抗議，而是經濟危機、地方權力重組、舊 MAS 網絡重新動員交疊在一起。Paz 可以改組內閣，但如果燃料、物價和地方封鎖沒有解，改組只能買時間。

秘魯大選決選也終於明朗。Keiko Fujimori 將在6月7日對上 Roberto Sánchez。問題是，第一輪後花了三十三天驗票，數千張選票被重新檢查，還有大約二十次廢票嘗試被駁回。這讓第二輪不只是左右對決，也是選民是否還相信選制的壓力測試。秘魯這幾年最缺的不是政治人物，而是能穩定撐住結果的制度信任。

幾條延續新聞簡短帶過：美伊停戰 MOU 看起來更接近，但伊朗否認已同意交出高濃縮鈾，真正的核查和制裁解除仍未解；台灣表示沒有收到美方暫停一百四十億美元軍售的通知，但川普把軍售放進全球彈藥調度與美中談判語境，台北的不確定感沒有消失；英國工黨黨魁危機也還沒收束，Keir Starmer 留在位子上，但 Andy Burnham 的挑戰路徑愈來愈具體；菲律賓 Sara Duterte 的彈劾審判已經開場，這會直接牽動下一屆大選布局。

中東今天的重點，是「停火」這個詞愈來愈需要小心使用。加薩方面，Board of Peace 說聯合國與夥伴的援助分發量比停火前增加超過七成，基本食物需求第一次穩住；Human Rights Watch 的批評是，這些數字沒有說出援助自二零二六年初以來下降，也沒有回到美以對伊朗開戰前的水準。換句話說，官方說比停火前多，人權團體問的是：有沒有達到人道最低需求。

黎巴嫩南部也是同一種矛盾。美方斡旋的停火還在紙面上，但以色列空襲和 Hezbollah 無人機攻擊沒有真正停下來。AP 報導，最近一輪以色列空襲造成十人死亡，包括急救人員與兒童；前一波攻擊則造成至少十九人死亡。停火如果只是降低全面戰爭風險，卻不能保護前線平民，那它在當地人生活中就很難被感覺成和平。

東亞的變化比較慢，但很關鍵。日韓領袖半年內第四次會面，李在明和高市早苗把能源安全放在很前面，包含 LNG 和原油協調。這不是突然和解，而是中東戰爭、荷莫茲海峽風險、北韓與中國壓力，把日韓推向更務實的合作。歷史問題沒有消失，但能源和安全供應鏈正在逼兩國建立新的工作關係。

日銀也進入6月會議前的敏感期。目前政策利率仍在百分之零點七五，下一次會議是6月15到16日。之前會議摘要顯示，有委員認為最早6月就可能升息。日本現在的難題是，如果太慢升息，日圓和進口成本會繼續有壓力；如果太快，企業和家庭的融資成本又會變成政治問題。

南亞與東南亞方面，印度和義大利把雙邊關係升級為特殊戰略夥伴，設定二零二九年雙邊貿易二百億歐元的目標，合作範圍包括 AI、國防、關鍵礦物、太空與 mobility。這其實是印度外交的一個縮影：它不是只找市場，也在找能共同生產、共同供應、共同抵抗風險的夥伴。

最後看太平洋。澳洲和斐濟在 Momi Bay 主辦太平洋跨國犯罪峰會，並舉行首屆區域警務首長會議。議題包括毒品走私、人口販運、洗錢和跨國犯罪網絡。這背後有很明顯的地緣政治味道：澳洲想把安全合作做得更制度化，斐濟和其他島國則想要資源，但也不想被單純當成大國競爭的棋盤。

接下來換到科技。

科技線今天最值得注意的，不是一個新模型，而是一份通諭。教宗 Leo XIV 將在5月25日與 Anthropic 共同創辦人 Dario Amodei 發表第一份關於 AI 時代人類尊嚴的通諭。這件事有一點不尋常，因為它把 AI 倫理從科技公司、政策圈和學界，拉進宗教與全球公共道德的場域。

你可以把它看成 AI 正在變成二十一世紀的工業革命問題。工業革命時，宗教和社會思想界會討論勞工尊嚴、城市貧困、資本集中；今天 AI 帶來的是勞動替代、知識生產、人格模擬、教育不平等，以及人類判斷被自動化系統包起來。梵蒂岡選擇和 Anthropic 同台，也是一種訊號：它把 AI 安全公司視為能代表風險討論的一方。

但這裡也有一個清醒的限制。通諭可以改變公共語言，不能直接變成模型審計。宗教領袖說人類尊嚴重要，跟企業真的願意揭露訓練資料、紅隊測試、系統風險與商業壓力，是兩回事。所以這件事的重點不是「教宗能不能管 AI」，而是 AI 的社會正當性正在被更廣的機構審視。

第二個科技重點，是 AI 晶片成本。Epoch AI 的資料顯示，記憶體，尤其是 HBM，已經接近先進 AI 加速器元件成本的三分之二。這代表 AI 擴張的瓶頸，不只是你能不能買到 GPU，而是你能不能拿到足夠的高頻寬記憶體、先進封裝與相關製造能力。

這件事會改變大家看 AI 基礎建設的方式。過去討論常常集中在「誰買最多 GPU」、「誰蓋最大資料中心」。現在問題更像供應鏈工程：HBM 供應商、CoWoS 封裝產能、電力、冷卻、網路交換設備，哪一環卡住，整個訓練和推論成本都會變。雲端大廠的資本支出不是單純炫富，而是被元件價格和產能約束推著走。

第三個科技重點在太空。SpaceX 發射了升級版 Starship，AP 說這是 NASA 登月架構仰賴的核心載具之一。Starship 的每次測試，外界通常很容易看成成敗二分：有沒有爆炸，有沒有回收，有沒有達到高度。但 NASA 關心的是另一件事：這套系統能不能從快速迭代，走到可驗證的任務可靠度。

這是 SpaceX 模型最有張力的地方。快速試錯能讓硬體進步很快，可是載人登月需要的不只是速度，而是程序、冗餘、驗證與可預測性。Starship 越接近 NASA 的正式任務，它就越不能只用創業公司的語言說故事，還要用載人航太的語言證明自己。

幾條 AI 和開發者延續新聞也值得放在一起。Google I/O 之後，Gemini 正在變成 Google 產品層的預設介面，從搜尋、Android 到 Workspace 都會更深地接上 AI。這對使用者是低門檻，對競爭者和監管者則是入口權力問題。當 AI 被塞進每個預設位置，使用者到底是在選擇 AI，還是在被帶進 AI，這會是接下來的爭議。

Hugging Face Transformers 5.9.0 則把 Cohere Command A+ MoE 等模型支援推進主線。這種版本新聞看似開發者小事，其實代表模型發布到工具鏈吸收的速度又變快了。模型公司可以推出新架構，開源工具鏈可以很快讓大家測，然後企業再把它放進評估流程。AI 生態系的迭代速度，就是靠這些無聊但關鍵的版本更新撐起來。

AI agent 這邊，Simon Willison 的 datasette-agent 0.1a4 把 agent chat 放進 Datasette 的 Jump to menu。這是一個小功能，但方向很明顯：agent 不再只是獨立聊天框，而是嵌進資料探索的工作流。當你在資料庫裡找表、查欄位、切資料，agent 直接在那裡出現，使用者就不必把問題搬去另一個介面。

openclaw 也有一個很生活化的 agent 更新：iMessage 審批可以用按讚或倒讚反應來允許一次或拒絕。這讓 agent 執行審批更像普通聊天，但也把安全問題拉得更近。因為越自然的操作，越容易被誤觸、被社交工程利用，或被錯的人在錯的情境下點掉。

加密貨幣今天有兩條和安全有關。第一條是 StablR 的 EURR 和 USDR 因多簽漏洞被攻擊者鑄出一千三百五十萬美元未抵押代幣，價格一度嚴重脫鉤。這類事件提醒我們，穩定幣的風險不只在準備金，也在權限管理。只要鑄幣權限被濫用，市場對「有抵押」的信任會瞬間失效。

第二條是 CoinDesk 提到，AI agent 可能把 crypto rails 推成機器對機器付款的基礎層。看多的人會說，鏈上支付、穩定幣和自動結算正好適合 agent；看空的人會問，如果 agent 付錯錢、被提示注入、或把授權交給惡意流程，誰負責追回？這會是 agent 經濟真正落地前必須回答的問題。

開發者社群今天也有兩個有趣訊號。Jujutsu 再次引發 Git 工作流疲勞討論，很多人不是討厭 Git，而是累於在大型 code review、長分支和堆疊變更裡維持乾淨歷史。另一篇 Go 轉 Rust 的遷移指南也升溫，說明系統語言選型不只是性能和安全，還包含團隊如何承受遷移成本。

最後，NASA 的 SpaceX CRS-34 補給任務把約六千五百磅物資送往國際太空站，其中近一噸是研究設備和樣本，支援大約五十項科學調查。這類新聞不如 Starship 華麗，但它代表另一種航太能力：穩定、重複、不中斷地把研究節奏維持住。國際太空站連續載人超過二十五年，靠的就是這種看似例行、其實很難的物流。

今天的世界像是在幾個斷點前停住：美伊可能停戰但還沒解核問題，烏克蘭戰場又看到高超音速武器，Ebola 疫情顯示公共衛生和社區信任不能分開。科技這邊，AI 則從模型競賽擴散到宗教倫理、晶片記憶體、太空運力和日常工作流。

你會發現，今天很多新聞都不是單點突破，而是系統承壓。外交系統、醫療系統、能源系統、供應鏈系統、軟體開發系統，都在被迫回答同一個問題：當壓力變成常態，什麼東西還能穩定運作。

以上就是今天的每日世界新聞。我們明天繼續追蹤。

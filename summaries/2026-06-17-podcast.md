EP105  6月17日 每日世界新聞與科技新聞

今天開場，我想先把你帶到三個很不一樣、但其實互相呼應的畫面。第一個，是 Évian 的 G7 會場，烏克蘭把防空需求重新推回主桌，美國則暗示俄羅斯石油制裁可能很快恢復。第二個，是中東，美伊框架看起來把荷姆茲航道從危機裡拉出來，卻又被以色列是否撤出黎巴嫩卡住。第三個，是剛果東部，Ebola 病例升到七百八十二例、一百八十一人死亡，但接觸追蹤只抓到五成多。

這三個畫面問的是同一件事：世界現在不是沒有協議、沒有制度、沒有技術，而是這些東西能不能真的穿過戰場、社群不信任和政治算計，落到現場。

先進入時事新事區。

今天最重要的外交線，是 G7 從中東危機重新轉回烏克蘭。Donald Trump 在 Évian 表示，既然荷姆茲海峽的石油流動恢復，美國就可以更快考慮恢復對俄羅斯石油運輸的制裁。這句話的重點不是語氣多強，而是它把兩個原本分開的危機接起來：中東航道一旦穩住，西方就少了一個不能動俄油的理由。

Volodymyr Zelenskyy 在 G7 的需求也很明確，就是更多 Patriot 防空。這不是抽象的軍援清單。前一天 Kyiv 才遭遇大規模飛彈與無人機攻擊，Kyiv Pechersk Lavra 這座 UNESCO listed monastery 也受損。對 Kyiv 來說，這些畫面有一個很現實的目的：把同情轉成攔截器、雷達、飛彈和後勤。

歐洲的態度更直接。英國宣布新一輪針對俄羅斯 shadow fleet 的制裁，法國和德國也推動美國更積極介入。可是 Trump 的位置比較複雜。他一方面說俄羅斯沒有贏，另一方面又保留自己作為調停人的空間。你可以把這理解成美歐之間的分工落差：歐洲希望把制裁當戰場壓力，美國則還想把制裁當談判籌碼。

第二條主線在中東。美伊框架原本像是今天少數可以讓市場和外交圈鬆一口氣的消息。它讓荷姆茲恢復通行，也把美國海上封鎖和伊朗核問題放進下一階段談判。但 Iran foreign minister Abbas Araghchi 今天把話講得更硬：任何美伊和平安排，都要以以色列從黎巴嫩撤出為條件。

這句話等於把 Lebanon front 正式塞回美伊談判桌。Iran 需要向 Hezbollah 和國內強硬派交代，不能讓美國只拿到航道穩定和核承諾，卻放任 Israel 在 Lebanon 保留安全區。Israel 這邊則完全相反。Benjamin Netanyahu 宣稱對伊朗取得歷史勝利，同時排除撤出 Lebanon，理由是 Hezbollah 的威脅還在。

所以你會看到一個矛盾：同一份和平框架，對美國來說是航道和核問題，對伊朗來說是全戰線停火，對以色列來說則不能犧牲前線安全自由。這也是為什麼 Trump 在 G7 對 Netanyahu 的 Lebanon 行動表達不滿。因為只要 Israel 繼續在 Lebanon 打，Iran 就有理由說協議沒有完整執行；只要 Iran 說協議不完整，美國就很難把這份框架包裝成真正的區域降溫。

第三條主線，是剛果東部 Ebola 疫情。剛果衛生部的數字是七百八十二例、一百八十一人死亡，致死率大約二成三，超過九成病例在 Ituri。這次是 Bundibugyo strain，不是過去比較常被討論、已有疫苗可用的 Zaire strain，所以應對工具少很多。

更麻煩的是接觸追蹤掉到五成六。這個數字很關鍵，因為 Ebola 防疫靠的就是快速找出接觸者、隔離、監測、讓安全葬禮不要變成下一輪傳播。但 Ituri 本來就有武裝衝突、大量流離失所、道路差，還有礦工和跨省移動。病例已經延伸到 North Kivu、South Kivu，甚至跨到 Uganda。也就是說，病毒不是在一個可控的醫療地圖上移動，而是在戰區、礦區和不信任的社群裡移動。

這裡最值得注意的是信任。Le Monde 的現場報導寫到，有地方醫療人員說，社區裡還有很多未通報死亡，也有人拒絕去醫療機構，因為他們不相信疫情或不相信外部醫療隊。Ebola 最怕的就是這個瞬間：公共衛生措施被理解成外來控制，安全葬禮被理解成搶走家人的身體，隔離被理解成再也回不來。

接著看歐洲內部治理。英國 Starmer 政府推動十六歲以下社群平台禁令，原本主軸是兒少安全、心理健康、霸凌和演算法傷害。但今天新的角度，是身障兒童倡議者警告，這種 blanket ban 可能切斷 disabled children 的支持網絡。對很多孩子來說，社群平台不只是娛樂，也是在現實生活找不到同儕時，找到角色模範和歸屬的地方。

這個辯論很難用支持或反對一刀切。支持方說，平台設計已經傷害兒少太久，政府必須畫線。反對方說，禁止孩子進入主流平台，可能只會把他們推向更地下、更難監管的空間，還讓大平台少承擔改設計的責任。真正的問題可能不是孩子能不能上網，而是平台能不能被迫降低成癮設計、限制陌生人接觸、處理演算法放大傷害。

東亞與東南亞今天的重點在南海。德國總統 Frank-Walter Steinmeier 訪問 Manila 時說，歐洲擔心 South China Sea 的緊張會像荷姆茲一樣威脅 freedom of navigation。他沒有直接點名中國，但他站在 Philippine President Ferdinand Marcos Jr. 旁邊提到二零一六年仲裁，意思已經很清楚。

這不是歐洲突然變成南海主要玩家，而是全球航道安全被重新打包。過去歐洲看 Indo-Pacific，常常是價值、規則、供應鏈分散。現在經過荷姆茲危機，航道被封的後果變得更具體：保險、能源、貨運、海軍部署，都會連到歐洲自己的經濟安全。北京對菲方官員 Gilberto Teodoro 的入境制裁，也讓南海爭端從海上執法延伸到個人制裁和外交羞辱。

台灣是延續線，今天沒有新的類別變化，所以只講一句更新。上週台灣以美製 HIMARS 朝台海方向實射，是第一次把這套 shoot-and-scoot 系統用在面向中國的實彈展示；新的觀察是，美國原本規劃增售八十二套 HIMARS，但在 Trump 和 Xi 會面後看似暫緩，台灣嚇阻能力和美中談判節奏被綁得更緊。

南亞方面，Pakistan-administered Kashmir 的抗議仍在升溫。JAAC 被禁後，警方逮捕支持者，爭議焦點包括電力、小麥補貼、服務改善，以及議會中十二個 Kashmiri refugee reserved seats。Pakistan defence minister Khawaja Asif 把抗議稱為 sedition 的開端，這讓原本偏民生和政治代表的訴求，迅速被國家安全語言包住。這種語言一旦啟動，談判空間通常會縮小，鎮壓成本則會上升。

印度則有一條氣候新聞。UNICEF 報告說，印度約三億九千兩百萬名零到十八歲兒童暴露在 extreme heat 之下，約佔兒童人口九成二，另有八千九百萬名兒童暴露於 heatwaves。這不是只關於天氣預報。孩子的身體調節能力更弱，學校、供水、醫療和戶外勞動安排如果沒有跟上，高溫會變成缺課、疾病和家庭收入壓力。

大洋洲有兩條。第一，澳洲 High Court 駁回政府在 indefinite detention damages claim 裡的抗辯，讓超過三百五十名 non-citizens 可能提出求償。這延續 NZYQ 判決後的憲法壓力，也把 mandatory detention 的成本從抽象人權爭議變成可能高達數千萬澳元的財政責任。

第二，Papua New Guinea 的 Launkalana 兒童因為二零一二年被洪水沖毀的橋一直沒有重建，必須游過有鱷魚的 Kemp Welch river 上學。UNICEF 把它放進兒童氣候風險報告裡，因為它說明 climate adaptation 不是只有海堤和電網，也是橋、路、學校和診所。當這些基礎設施失效，氣候風險就會變成每天上學路上的風險。

時事延續區簡短收束幾條。Colombia 決選前，EU observers 已經否定 Petro 的 fraud claims，De la Espriella 和 Cepeda 現在把安全政策變成最大分歧；Australia-Solomon Islands 安全重設繼續往 comprehensive treaty 和 policing cooperation 走；全球兒童氣候報告則指出，超過十億名兒童同時暴露於至少三種 climate hazards，Sahel 有超過四百萬兒童面對熱浪、極端高溫和沙塵暴三重威脅。

轉到科技。今天科技線的核心不是單一超大產品發布，而是「誰能把複雜系統變得可信任」。local LLM、AI agent、tokenized finance、具身智能、資安 phishing，全部都在問同一件事：當工具變強、鏈條變長，你到底信任哪一層。

科技新事區第一條，是 Ollama v0.30.9-rc2。這次更新讓 context shift 支援 shiftable prompts。聽起來很底層，但對本機 LLM 長對話很重要，因為你要讓模型在上下文移動、提示重排、記憶體壓力之間維持可用。Ollama 的價值一直不是只在模型列表，而是在把 llama.cpp、模型格式、provider 行為和本機推論包成比較容易使用的 runtime。

不過這仍然是 release candidate。對個人實驗，可以很快試；對 production agent，最好先看推論結果、工具呼叫和資源消耗是否有微妙變化。agent 系統最可怕的不是完全壞掉，而是行為只偏一點點，卻在自動化流程裡累積成錯誤。

第二條，是 PyTorch XPU CI release。`ciflow/xpu/186797` 修掉兩個 downstream CI failures，焦點包括 mm_plus_mm Triton disable 的測試邊界，以及 XPU 在 dynamic shape GPU wrapper 裡的 xfail 註冊問題。這不是一般使用者應該立刻升級的正式版，但它代表 AI framework 的維護重心正在變。過去大家習慣把 GPU 等同 CUDA，現在 XPU、ATen fallback、多硬體測試矩陣都要被照顧。

這類新聞很容易被忽略，但它其實決定了 AI 工具鏈能不能真的跨硬體。模型很大、benchmark 很漂亮，如果底層 kernel、compiler path 和測試邊界不穩，開發者最後還是會回到少數最穩的硬體組合。

第三條，是 Coinbase 的 System Update。Coinbase 把 AI-powered advisor、global unified liquidity、options trading 和一比一 backed tokenized US stocks 放在同一波更新裡。這很像加密交易所想往全資產 brokerage 走：不只是買賣 crypto，而是把股票、選擇權、跨境流動性、AI 輔助決策包進一個介面。

看多方會說，這是傳統金融和鏈上資產融合的下一步；看空方會問，AI advisor 真的能被監管允許到什麼程度，tokenized stocks 的股息、保管、投票權和跨境銷售怎麼合規。尤其是 AI 投資建議，責任邊界很難模糊。使用者以為只是聊天，監管機關可能會認為那就是建議。

加密貨幣還有幾條值得看。Pump.fun activity 三個月跌了八成，PUMP token 六個月跌四成，也拖低 Solana fee。這說明 meme coin 週期退潮時，鏈上收入品質會被重新檢驗。Squid 接入 Ripple 的 RLUSD stablecoin，代表合規 stablecoin 想進入更多 cross-chain swaps；但穩定幣真正的難題是流動性深度和使用需求，不是掛上更多鏈就結束。

Bitcoin DeFi 的報導也很有意思。DeFi total value locked 在不到一年內從約一千八百億美元降到七百億美元，Bitcoin DeFi demand 被形容為 concentrated in small but deep pockets。這句話很精準：需求不是沒有，但不是全民爆發，而是少數資本深、願意承擔橋接和智能合約風險的使用者在試。

前沿科技今天看中國具身智能。大曉機器人說 Kairos 3.0 在四項全球具身智能 benchmark 取得 SOTA，訓練資料擴展到一百萬小時，約為傳統真機遙控採集模式的十倍。這條新聞真正重要的地方，不是又一家公司說自己做 world model，而是中國具身智能競爭正在從「做一台像人的機器人」轉向「誰有資料、誰能預測物理後果、誰能端側部署」。

世界模型之所以被反覆提，是因為機器人不能只辨識物體。它要知道推一下會怎樣，抓錯會不會掉，水、光線、遮擋、摩擦會怎麼改變下一秒。可是真實部署的風險也在這裡。benchmark SOTA 不等於酒店、無人零售、無人物流的長期可靠性；資料量變大，也不等於維修、保險和客戶導入會變簡單。

消費硬體有 OBSBOT 尋影。它在高端 Webcam 市場市占率超過五成，連續五年每年成長五成。這是一個很好的提醒：AI hardware 不一定都要做手機或機器人。遠距辦公、直播、會議、自動構圖和智慧追蹤，也可以長出一個高價細分市場。問題是，這個市場會持續受旗艦手機、筆電內建鏡頭和軟體虛擬攝影棚擠壓。

資安方面，EvilTokens 值得注意。這個 phishing kit 不是仿冒登入頁，也不是直接偷密碼，而是濫用 Microsoft legitimate authentication flow，讓攻擊者取得帳號存取。這對企業防禦很麻煩，因為傳統教育使用者看網址、看假頁面的效果會下降。你要看 OAuth consent、device code flow、異常 token 使用、條件式存取，以及有哪些 app 被授權。

換句話說，帳密不是唯一邊界。token、授權、session 和第三方 app governance 才是下一層攻擊面。越多公司導入雲端身份和單一登入，這層邊界就越重要。

科技延續區也收幾條。Fable 5 出口管制爭議仍在延燒，新的焦點是資安社群質疑，美國把協助修補漏洞的能力也當成危險模型能力，可能傷到 cyber defense。OpenClaw 2026.6.8 延續把 Telegram、WhatsApp 和 CLI delivery 做成 runtime 一級能力的方向，但權限、審計與重送語義會越來越重要。langchain-openai 1.3.2 是小版維護，提醒 production agent 的 reproducibility 成本仍在。

今天科技我不做深度區，因為沒有真正符合旗艦級產品發布、十億美元融資或大規模資安事件的類別變化。比較值得帶走的是一個橫向觀察：AI 和 crypto 都在從「能力展示」進入「責任邊界」階段。模型能不能跑、交易能不能做、agent 能不能發訊息，都不是終點；真正的問題是誰負責、怎麼回滾、怎麼審計、怎麼證明它沒有在錯的時間做錯的事。

今天結尾，我想把時事和科技拉回同一句話。G7 需要把制裁、防空和外交承諾落到戰場；美伊框架需要穿過 Lebanon 這個破口；Congo Ebola 需要穿過社區不信任；科技產品則需要穿過版本、權限、token 和合規邊界。6月17日這一天的共同主題，就是「可執行性」。世界不缺漂亮文本，也不缺強大工具，缺的是它們在混亂現場仍然可信、可查、可持續。我們明天繼續。

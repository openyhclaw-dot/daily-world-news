第99集，6月11日。

今天這集，我們先看美國和伊朗。荷姆茲周邊的互擊，已經不只是談判前的壓力測試，而是開始逼近一個更危險的問題：雙方還有沒有能力把軍事行動重新放回外交框架裡。第二條主線，是 Ebola 疫情從剛果和烏干達，延伸成美國、歐洲和非洲之間的旅行管制爭議。後半段轉到科技，今天的共同問題是：AI 和加密系統越會替人行動，責任邊界就越難畫。

先從時事開始。

美國再度空襲伊朗，是今天最重要的事件。AP 報導，美軍攻擊伊朗南部防空和雷達設施後，Trump 威脅說，如果 Tehran 不接受和平條件，美方會有更多軍事行動。伊朗隨後用飛彈和無人機攻擊 Bahrain、Kuwait、Jordan 等地的美軍相關基地。這不是一次孤立報復，而是兩個月脆弱停火之後，雙方重新把戰場推到談判桌前面。

真正危險的地方，是美國和伊朗的條件完全錯開。美國要求 Iran 限制濃縮鈾、飛彈計畫和對 Hezbollah 等代理人的支援，並且想在協議簽署前保留制裁和凍結資產作為槓桿。伊朗則要求解除制裁、處理凍結資產，還要把 Lebanon 戰線放進同一套停火安排。美國想把 Lebanon 切開，Iran 想把 Lebanon 綁回來。這個差異如果不解決，軍事降溫就很難靠一句「我們還在談」來維持。

Qatar 代表團趕到 Tehran 斡旋，代表還有人想把局面拉回外交。但調停者能做的，是創造說話空間；他們不能替雙方承擔國內壓力。Trump 需要證明自己沒有被 Tehran 拖延，Iran 也需要證明自己不是在空襲後被迫簽約。這讓每一次小規模軍事行動，都可能變成下一次升級的理由。

荷姆茲在這裡不是背景板。它是能源通道、軍事通道，也是政治象徵。美國會說航行自由和基地安全不能被 Iran 挾持；Iran 會說美軍靠近海峽和封鎖行動本來就是侵犯主權。當雙方連同一件事的語言都不同，風險不只是攻擊本身，而是每一方都會把對方的動作翻譯成更惡意的版本。

Lebanon 戰線也在牽動這一局。Israel 空襲 Tyre，造成至少八人死亡、三十二人受傷，並且首度要求 Tyre 的歷史基督教區撤離。Tyre 不是一般邊境村落，它是歷史古城，也是宗教和文化記憶很密集的地方。Israel 說 Hezbollah 滲透民用區，使用的是軍事必要性語言；Lebanese 社會抗議，回應的是平民生活和文化城市被推上戰場的恐懼。

更麻煩的是，Lebanon 官方談判桌和 Hezbollah 的戰場行動之間，本來就有距離。正式文件如果約束不到真正開火的一方，停火就會變成很脆弱的儀式。Tyre 今天不只是單一空襲，而是停火架構失效的一個訊號。

同一天，西方六國對 West Bank 極端 settler 網絡祭出制裁。Australia、UK、Canada、France、Norway、New Zealand 都把被指涉支持 settler violence 和 settlement expansion 的個人、企業或官員列入制裁。這不是最大規模的制裁，但政治訊號很清楚：West Bank 的定居點金融和暴力支持網絡，開始被一些盟友當成可以單獨懲罰的對象。

Israel 右翼會說，這是盟友誤解安全現實；歐洲和英聯邦國家則會說，settler violence 正在破壞 Palestinian civilians 的安全，也讓兩國方案更不可行。真正的變化是，Gaza、Lebanon、West Bank 這幾個政策抽屜正在被打開放到同一張桌上。

接著看 Ebola。AP 報導，美國國務卿 Marco Rubio 要求歐洲對來自剛果和烏干達疫區的旅客採取更嚴格措施，理由是歐洲每天有三百多班航班連往美國，而且 World Cup 將增加大量跨境人流。美國已經禁止過去三週到過疫區的人入境，並承諾投入超過兩億美元抗疫；歐盟也把應變資金提高到大約三千六百萬美元。

但公衛問題一旦進入旅行限制，就會變得很政治。美國看的是輸入風險，尤其是大型賽事和轉機網絡；歐洲通常會對全面旅行限制更謹慎，因為過度封鎖可能讓人隱瞞旅史，也可能打擊疫區經濟。剛果和烏干達現場最需要的，仍然是診斷、隔離、接觸者追蹤、醫療人員保護和社區信任。旅行管制可以降低部分風險，但如果它取代現場投資，疫情不會因此被解決。

這次剛果東部疫情的難點，是 Bundibugyo 型 Ebola。它不像比較常被討論的 Zaire 型 Ebola 那樣有成熟疫苗工具。加上 Ituri 一帶有武裝衝突、礦工流動、位移人口和對外來醫療隊的不信任，所以病毒的路線不是只有家庭和醫院，也包含逃難路線、工作路線和戰爭經濟。

非洲另一條線是 Ethiopia。上週大選多數地區完成投票，但 Oromia 和 Amhara 的安全問題讓一百四十三個投票站受干擾，Tigray 連續第二次沒有參與。超過五千萬選民登記投票，Abiy Ahmed 的 Prosperity Party 仍被看好取得多數。對政府來說，選舉展示國家仍可運作；對反對派和人權團體來說，問題是政治自由、媒體空間和衝突地區的代表性。

南亞方面，Pakistan-administered Kashmir 又出現新事件。AP 報導，一架 Pakistan army MI-17 直升機在 Muzaffarabad 附近墜毀，軍方說是技術故障，機上軍事人員全數死亡。單看這則，它是軍機事故；但放進背景，就會變得敏感，因為直升機載運的 Rangers，是在 JAAC 罷工抗議和地方安全緊張下部署的。

JAAC 被禁，網路暫停，Rawalakot 先前的致命衝突還沒降溫，帶頭者又面臨 sedition charges。地方抗議的核心，是十二個保留給 Kashmir refugee 的議席。支持者說，這些席次讓不住在當地的人擁有過度政治影響；政府說，這是憲法保護安排，也牽涉更大的 Kashmir 問題。當政治代表權爭議被放進安全鎮壓和軍事部署裡，即使墜機不是抗議造成的，社會感受也會把它和緊張局勢連在一起。

台灣今天的訊號，是美製 HIMARS 實彈演訓。報導稱，台灣在西岸以 HIMARS、Paladin howitzers、反戰車飛彈和 Thunderbolt-2000 進行反登陸演練，強調發射後快速轉移。戰術意義是提高機動打擊和反登陸可信度；政治意義則更敏感，因為它發生在美國對台軍售和 Trump 是否與 Lai Ching-te 通話的脈絡中。

支持者會說，台灣如果要提高嚇阻，就必須讓對岸相信登陸成本很高；降溫派會擔心，如果這些軍事訊號和高層通話、軍售審查疊在一起，北京會把它解讀成政治升級，而不只是例行訓練。台海的難點就在這裡：同一個動作，在台北看是防衛，在北京看可能是挑釁，在 Washington 看可能是談判籌碼。

North Korea 是延續更新。Xi Jinping 訪問 Pyongyang 時沒有公開強調 denuclearization，這件事還在被 Seoul、Tokyo、Washington 解讀。它不代表 Beijing 突然支持北韓核武，但代表中國至少在公開場合把「去核」優先順序往下放。對 Kim Jong Un 來說，這就是外交上的勝利，因為他一直想讓外界接受 North Korea 已經是核武國家的現實。

美洲的兩條線，一條是 Colombia，一條是美國通膨。Colombia 6月21日決選，由 Iván Cepeda 對上 Abelardo de la Espriella。Cepeda 延續 Gustavo Petro 的 Amazon 保護、停止新油氣探勘和原住民參與路線；de la Espriella 則主張 fracking、化石燃料開發和更強硬治安。這不是單純左派和右派，而是 Colombia 要不要繼續把自己定位成氣候政治前線國家。

美國通膨則重新被中東能源衝擊推上來。Guardian 市場即時報導稱，美國 5月 CPI 年增四點二個百分點，為 2023年4月以來高點，其中汽油上漲七個百分點，貢獻超過六成的月增壓力。若荷姆茲局勢再惡化，政策制定者會同時面對成長放緩、物價回升和政治壓力。

大洋洲今天除了移民拘留判決，還有 Solomon Islands。Australia 正在推動 Solomon Islands 加入 regional policing initiative，希望降低它對中國 security assistance 的依賴。Matthew Wale 新政府則要處理燃料、醫療和災後復原需求。這裡不是簡單的「倒向誰」，而是 Pacific 小國如何在 Australia、China、New Zealand、United States 之間保留主權空間。

比較短的幾條延續更新一起帶過。Ukraine 方面，Starmer 邀 Zelenskyy、Macron、Merz 到 London 討論支援和安全保證；Philippines 和 Vietnam 的 South China Sea 合作仍在延續；USMCA 二零二六年檢視期逼近，北美企業等待三國能不能給出更長供應鏈確定性；Australia 的 AI regulation 也進入內政議程，資料中心、科技稅和社區回饋都開始被放上桌。

從時事看下來，今天的共同主題是邊界。美伊在測試停火和戰爭的邊界，Ebola 在測試公衛和旅行自由的邊界，台灣在測試防衛演訓和政治訊號的邊界，Australia 在測試行政權和人身自由的邊界。接下來轉到科技，你會發現科技新聞也在問同一件事：當系統更會替人行動，人的控制線到底畫在哪裡。

科技第一條，是 Transformers v5.11.0 納入 DiffusionGemma。這不是一般意義上的又一個聊天模型，而是把 diffusion-style generation 放進文字模型工作流，試圖降低傳統自回歸推論的速度瓶頸。Simon Willison 也提到，Google 先前的 Gemini Diffusion preview 曾展示過每秒八百五十七 tokens 的速度。這個數字很吸睛，但速度展示不等於所有任務都更可靠，也不等於長文推理、工具使用和事實校驗突然被解決。

DiffusionGemma 真正值得注意的，是它讓更多開發者可以在 Hugging Face 生態裡實驗非自回歸文字生成。這可能帶來速度和批次生成上的優勢，但也會帶來新的評估問題：如果模型不是線性生成，我們要怎麼觀察它在哪一步出錯？要怎麼控制輸出一致性？要怎麼讓它安全地接工具？

第二條，是 Tether 領投 Neura robotics 最高十四億美元融資，並計畫把 crypto wallet integration 放進機器人生態。這件事把穩定幣資金從金融基礎設施推到實體 robotics。看多的人會說，這是穩定幣公司把資本投入 AI 和機器人實體世界；看空的人會問，硬體交付週期很長、估值很重、監管也更複雜，錢包整合到底是產品價值，還是只是把加密敘事貼到機器人上。

這條新聞跟今天時事其實有呼應。當資本從鏈上金融走向機器人，它不只是投資另一個產業，而是把支付、身分、授權和實體行動接在一起。軟體錯了，可能是交易失敗；機器人錯了，可能撞到東西、傷到人、或在現場做出錯誤操作。

第三條是 Microsoft 六月 Patch Tuesday。KrebsOnSecurity 指出，這次 Microsoft 修補接近兩百個安全漏洞，是單月修補量的高點之一。對一般使用者來說，這聽起來像例行更新；對企業資安團隊來說，這代表優先排序的壓力很大。你不可能用同樣速度修所有漏洞，所以重點會落在已知遭利用漏洞、暴露在外的服務、權限提升和能被勒索軟體利用的路徑。

Krebs 也追蹤 The Gentlemen ransomware group。這個勒索團體正透過招募和 affiliate 生態擴張，成為以受害者數計算相當活躍的組織之一。企業防禦不能只盯 malware family，還要盯初始入侵代理、資料外洩勒索、受害者談判和加密貨幣洗錢路徑。勒索軟體已經不是單一技術問題，而是一整個犯罪供應鏈。

LLM 法律責任也有新訊號。Gary Marcus 關注 Google 幻覺責任判例，重點是，如果搜尋或 AI 回答產生錯誤，而且造成實質損害，平台能不能只靠條款免責。過去很多 AI 風險討論停在「模型可能會錯」；現在問題變成「模型錯了之後誰賠」。越多 LLM 被包進搜尋、客服、醫療、金融或法律流程，錯誤就越不可能只被看成產品瑕疵。

OpenAI 公布 PRC-linked influence operations 報告，也把 LLM 安全拉到政治資訊戰。報告說，相關行動使用 AI 影響美國科技辯論、資料中心敘事、關稅和 ChatGPT 相關假訊息。這裡需要兩個態度同時存在。一方面，AI 確實讓低成本大量生成政治內容更容易；另一方面，外界仍然需要可驗證樣本和跨平台資料，才能判斷影響規模。

AI agent 的部分，Mastercard 推出 Agent Pay for Machines，把 autonomous agent 支付接到穩定幣場景。這比一般 API 工具調用更敏感，因為它牽涉付款。當 agent 可以代表機器或服務自動付款，授權邊界、退款、詐欺、KYC 和責任歸屬都會變難。你可以想像一個工廠設備自動購買零件、一個軟體 agent 自動租用算力，效率很高；但如果它被 prompt injection 或錯誤資料誘導付款，誰有權撤回，誰負責損失？

LangChain Perplexity 1.4.0 支援 bind_tools 和 Responses API tool round-trip，這是比較工程層的更新，但方向一致：搜尋型模型被放進 agent tool loop。OpenClaw 2026.6.10-alpha.1 也是 agent runtime 的小步迭代。這些框架更新不會像新模型發布那樣吸睛，但它們決定 agent 能不能被觀察、重試、回滾和審計。

加密貨幣今天有幾條值得合併看。Bitwise CIO Matt Hougan 說，財務顧問對 stablecoins 和 tokenization 的興趣，正在超過 Bitcoin 本身。這代表加密敘事從「買 BTC」轉向「把金融資產搬上鏈」。支持者會說，這能吸引更穩定的機構資金；反方會問，tokenized assets 的法律所有權、託管和二級流動性是不是真的清楚。

另一條是 SpaceX IPO 可能成為 Bitcoin 短線資金逆風。如果 SpaceX IPO 預留最高三成股份給散戶，一部分資金可能從 BTC、ETH 等風險資產轉出。這不是單一利空，而是資金在不同高風險敘事之間輪動：昨天是 crypto，今天可能是 SpaceX，明天又可能回到 AI。

Raydium 則有一個具體資安事件。它一個 retired AMM program 被 exploit，約一百三十四萬美元從五個 inactive liquidity pools 流出，treasury 將補償受影響用戶。這不是系統性崩盤，但很能說明 DeFi 的技術債。就算主產品已經遷移，舊合約、退休程式、被遺忘的 liquidity pool 仍然可能成為攻擊面。

Solana 生態的 Helius 收購 Light Protocol，則把 onchain privacy 推向整併期。支持者會說，錢包和應用層需要更好的隱私，否則所有交易都變成公開行為紀錄；監管方會說，隱私不能變成洗錢和制裁規避。真正的技術挑戰，是怎麼做選擇性揭露；真正的政治挑戰，是誰有權要求你揭露。

前沿科技這邊，Dario Amodei 發表 AI exponential 政策文章，主張治理速度要跟上能力曲線。支持者會說，frontier lab 願意談政策是一件好事；批評者會要求更清楚的外部審計、能力揭露和可執行監管。JPL 維持十三年 Curiosity rover 科學任務，則提醒我們，真正的技術能力常常來自長期維護，而不只是發布當天的展示。

最後是 consumer tech。Apple WWDC 後的 AI 評價仍然分歧，市場擔心 Siri 回歸時間表、地區可用性和 monetization。這不是說 Apple 沒有進展，而是手機 AI 競爭已經從硬體規格轉向長期軟體能力。攝影機、瀏覽器、密碼管理器、相簿和 Shortcuts 都可能變成 AI 入口，但每個入口都牽涉隱私、權限和責任。

今天科技段可以用一句話收束：AI 和 crypto 都在從「給你工具」走向「替你行動」。Transformers 讓新生成路線更容易被測試，Mastercard 想讓 agent 付款，Tether 想把錢包放進 robotics，DeFi 想把信用和資產搬上鏈。這些方向都有真價值，但共同風險是，行動權越大，錯誤成本越高。

回到整集主題，今天的世界和科技其實都在談同一件事：邊界正在被重新畫。國家之間是軍事和外交的邊界，公共衛生是旅行自由和集體安全的邊界，AI agent 是授權和責任的邊界，加密系統是隱私和合規的邊界。接下來要看的，不只是有沒有新的攻擊、新的疫情、新的模型或新的融資，而是制度能不能把邊界重新畫得讓人理解、也讓人追責。

今天就到這裡。我們明天繼續。

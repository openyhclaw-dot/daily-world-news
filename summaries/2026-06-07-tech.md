## 🤖 AI Agent

• 🔥13 **OpenClaw v2026.6.5-beta.1 修掉 reasoning scaffolding 外洩到原生通道的風險** — OpenClaw beta 版提到 QQBot 會在 native delivery 前剝除 model reasoning/thinking scaffolding，避免 raw thinking 標記進入頻道回覆；同時把 MCP tool result 的 resource_link、audio、malformed image 等型別 coercion 補強。這是 agent runtime 的實用安全修補，但也提醒多通道 delivery 越多，輸出邊界越需要測試。

  📎 來源：[GitHub](https://github.com/crewAIInc/crewAI/releases/tag/1.14.7a2)

• 🔥10 **Simon Willison 發布 MicroPython WASM sandbox，服務 Datasette Agent code execution** — micropython-wasm 0.1a2 加入 CLI，搭配文章說明如何用 WebAssembly 與 MicroPython 做 Python code sandbox，目標是給 Datasette Agent 一個受限執行環境。這對 agent tooling 很實際；批評點是 sandbox 仍要面對資源限制、逃逸風險與套件生態不足。

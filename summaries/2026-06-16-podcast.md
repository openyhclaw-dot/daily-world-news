EP104  6月16日 每日世界新聞與科技新聞

今天開場，我想先把你帶到兩個同時發生、而且彼此牽動的畫面。第一個，是荷姆茲海峽可能重新打開，因為美國和伊朗終於把一份六十天框架協議擺上桌；第二個，是 Kyiv 的修道院在夜間空襲後起火，烏克蘭又一次把「防空」這個字，推到 G7 領袖面前。這兩件事看起來一個在中東，一個在歐洲，但背後問的是同一件事：大國現在還有沒有能力，把軍事衝突翻譯成可執行的政治安排。

先從時事的新事區開始。

美伊這份六十天框架協議，是今天最重要的一條線。協議的核心是：停火延長，荷姆茲海峽分階段重開，美國的海上封鎖放鬆，接著再談伊朗核計畫、制裁 relief，還有一部分被凍結資產的處理。這裡的數字很關鍵，荷姆茲海峽大約承載全球五分之一的石油和液化天然氣流量，所以只要船能動，能源市場就會先鬆一口氣。

但你不要把這件事理解成「中東和平突然來了」。它比較像一個止血包，先壓住能源航道和美伊正面衝突，再把真正難的問題往後推。伊朗強硬派不滿的是，協議沒有給足制裁解除和戰爭賠償的保證；美國這邊則把它包裝成 Trump 用壓力換來航道重開。更麻煩的是 Israel 並不在這張桌子的核心位置。Israel 防長 Israel Katz 已經說，Israel 不會撤出 Lebanon、Syria 和 Gaza 的安全區，而且如果 Iran 攻擊，Israel 會全力反擊。

所以這份協議最大的矛盾，是能源市場需要的是可預期，區域盟友要的是安全保證，而談判文本現在只先滿足第一個需求。船可以先走，但 Hezbollah、Lebanon、Gaza、Iran nuclear program，這些都還留在後面。

第二條主線在 Ukraine。6月15日夜間，Russia 對 Ukraine 發動大規模飛彈和無人機攻擊，Kyiv Pechersk Lavra 的 Dormition Cathedral 起火，文化設施也受損。外媒引述的數字是，Russia 一夜間動用七十枚飛彈和六百多架無人機，至少五人死亡。Ukraine 總統 Zelenskyy 把這件事直接帶到 G7，說這不只是軍事攻擊，也是對文化遺產和宗教象徵的攻擊。

Russia 的說法照舊是否認，甚至把修道院受損歸因於 Ukraine 防空。可是這場敘事戰對 Kyiv 來說有一個現實目的：它要 G7 不只譴責，而是交付更多反彈道防空。這是 Ukraine 這一年反覆碰到的困境。它可以證明自己被打，也可以把被打的畫面推到世界面前，但如果防空彈藥、攔截系統、後勤沒有跟上，國際同情不會自己變成防護網。

第三條主線，是 Congo 東部 Ebola 疫情突然變得更危險。剛果衛生部到 6月14日的數字是七百八十二例，一百八十一人死亡，致死率大約二成三，超過九成病例集中在 Ituri。這次麻煩在於病毒型別是 Bundibugyo，目前沒有核准疫苗或治療；接觸追蹤有效率掉到五成多，而且病例已經擴到 North Kivu、South Kivu，還跨進 Uganda。

你可以把它想成三層危機疊在一起。第一層是醫療，沒有對應疫苗，檢測和隔離就更重要。第二層是治安，Ituri 本來就有武裝衝突和大規模流離失所，醫護人員被攻擊不是抽象風險。第三層是信任，過去幾天已經出現錯假訊息和醫療設施遇襲。Ebola 防疫最怕的不是只有病毒傳播，而是地方社群開始把醫療隊也看成威脅。

接著看 G7。這次峰會在法國 Évian-les-Bains，時間是 6月15到17日，議程把 Iran 戰後能源、Ukraine 防空、Gaza 停火、對中國產能和貿易失衡的共同立場全部放到同一張桌上。這也說明 G7 現在更像危機管理會議；真正的問題不是能不能寫共同聲明，而是能不能把聲明翻譯成軍援、金融、制裁和能源安排。

歐洲還有一條科技治理新聞。英國 Starmer 政府宣布，最快 2027 年春天，十六歲以下將不得使用 TikTok、Snapchat、Instagram、YouTube、Facebook、X 這些主要社群平台；十八歲以下也會被禁止使用模擬親密關係的 romantic chatbots。政府說，九成受訪父母支持提高社群平台年齡門檻。

這個政策的直覺很容易理解，家長希望把孩子從演算法、霸凌、性化內容和陌生人私訊裡拉出來。但平台公司也警告，如果主流平台被一刀切禁止，青少年可能轉去更地下、更不受監管的地方。真正的問題是，英國到底是在管兒童，還是在管平台的設計責任。

東亞方面，Xi Jinping 結束近七年來第一次訪問 North Korea。這趟 Pyongyang 行，官方講的是友誼、貿易、農業、建設、科技和跨境交通重啟；比較少講的，是 denuclearization。這個沉默本身就重要。Beijing 要重新拉住 Pyongyang，因為 North Korea 這兩年靠 Russia 越來越近；Kim Jong Un 則需要中國這個舞台，證明自己不是只剩 Moscow 一個選項。

這對日本、韓國和美國都不是小事。如果 Beijing 對 North Korea 的核地位變得更能忍受，東北亞的嚇阻結構會更硬。South Korea 又還卡著 Yoon Suk Yeol 的 martial law、insurrection 和 drone case 延伸調查，安全部門與 presidential power 的邊界沒有完全收束。

台灣這邊，HIMARS 實射是延續故事，但還值得留一句。上週台灣在台中沿岸以美製 HIMARS 朝台海水域實射，展示 shoot-and-scoot 的反登陸能力。今天沒有新的類別變化，所以不重講背景。新的觀察是，美國原本規劃再售八十二套 HIMARS，但在 Trump 和 Xi 會面後看似暫緩，這讓台灣嚇阻能力和美中外交節奏被綁得更緊。

南亞與東南亞的短線更新，是 Pakistan-administered Kashmir 抗議仍在外溢，Balochistan Liberation Front 的 Allah Nazar Baloch 公開支持 resistance；Philippines 這邊，北京對 Teodoro 的制裁口水戰沒有升級成新事件，重點仍是 South China Sea 爭端開始從海上執法延伸到個人制裁。

中東延續區也短講。Lebanon 是美伊框架最大外部破口，因為 Israel 說不撤，Hezbollah 還沒正式給出能讓人放心的回應；Gaza 停火、Hamas disarmament 和 reconstruction 仍被 G7 外交綁在一起，但沒有新協議。Ethiopia 大選仍受 Oromia、Amhara 安全問題拖累，沒有最終類別變化。Australia 的 indefinite detention 案則有比較清楚的新後果，超過三百五十名 non-citizens 可能因 High Court 判決取得賠償，這會把 Australia 移民拘留政策的法治成本具體化。

最後補一個全球風險。NOAA 確認 El Niño 形成，而且模型給出六成三機率，會在 11月到1月達到非常強的級別，可能躋身 1950 年以來最強事件之一。這不只是天氣新聞。當世界同時有戰爭、能源波動、糧食價格壓力和人道援助削減，強 El Niño 會把乾旱、洪水、野火和糧食安全風險一起推高。

時事的深度區，我想回到美伊框架。它之所以重要，不是因為它已經解決中東，而是因為它展示了現在外交的最低可行版本：先保航道，再保油價，最後才談秩序。

這個順序很現實，也很危險。現實在於，沒有哪個政府能長期承受全球五分之一油氣航道不穩；危險在於，當談判優先處理市場的急性疼痛，政治創傷就會被留在體內。Iran 會問制裁和資產，Israel 會問 Hezbollah 和安全區，Gaza 會問戰後治理和重建。這些問題沒有消失，只是被六十天的技術談判包起來。

所以接下來兩個月，你可以看三個訊號。第一，Hormuz 的商船和保險成本是不是真的回到正常。第二，Israel 在 Lebanon 的軍事存在有沒有收縮。第三，Iran 核談判是否只是在拖延，還是出現可驗證安排。如果這三個訊號只有第一個改善，那協議就是能源停火，不是區域和平。

說完時事，我們轉到科技。今天科技線的共同主題，是「基礎設施的可信任程度」。LLM runtime、Windows 修補、crypto treasury、具身智能和生醫搜尋，看起來分散，其實都在問同一件事：當一套系統變得越來越大，你到底信任哪一層。

科技新事區第一條，是 Ollama v0.30.9 release candidate。這次主要更新，是把底層 llama.cpp 推到 b9637。這不是那種會讓普通使用者驚呼的旗艦模型發布，但對每天跑本機模型的開發者很重要，因為 local LLM 的痛點常常不是模型不夠炫，而是 provider、模型格式、推論 backend、記憶體行為之間磨來磨去。Ollama 的價值就在這裡：它把很多底層摩擦包起來。

可是我會提醒你，這是 rc 版本，不是穩定版。對個人實驗可以很快升，對 production agent 則應該先在測試環境看推論結果、工具呼叫和資源消耗有沒有變。Agent 系統最怕的不是一次崩掉，而是行為微妙變了，卻沒有人發現。

第二條，是 Hugging Face Transformers v5.12.1。這個 patch release 修補 PEFT 最低版本，並修正安裝 mistral-common 時，auto tokenizer 對 Mistral tokenizer 的解析問題。聽起來很小，但 tokenizer 錯配會讓模型輸入和輸出都變得不可靠。對研究者，它影響可重現性；對模型服務商，它可能影響線上結果一致性；對企業，則是升級版本時的 hidden cost。

這也是今天 LLM 生態的一個縮影。大家談 frontier model，很容易只看能力；但每天讓團隊花時間的，常常是 dependency pin、tokenizer、adapter、backend 和 benchmark 重跑。

第三條，是 Windows 安全更新。Microsoft 六月 Patch Tuesday 修補約兩百個漏洞，其中有 BitLocker bypass、本機權限提升，還有三個被公開討論的 zero-day。後續研究者 Nightmare-Eclipse 又釋出 RoguePlanet 和 GreatXML 兩個 PoC，把 Windows Defender、Volume Shadow Copy、Recovery Environment 這些邊界都拉進討論。

對企業端，這不是只看 CVSS 的時候。你要看哪些機器有 BitLocker，誰可以插 USB，ISO mount 是否被限制，六月修補是不是真的落地。這場 disclosure 衝突的教訓是：當研究者和廠商關係惡化，漏洞資訊本身就可能變成攻擊者的時間優勢。

AI agent 這邊，OpenClaw v2026.6.15-alpha.1 延續快速迭代。它的重點不是單一功能，而是把 Telegram、WhatsApp、CLI backend、rich text delivery 這些通訊渠道，逐步納入 agent runtime 的核心能力。這很符合 agent 產品化的方向。真正有用的 agent 不只是會想，還要能在正確渠道，把正確格式，送到正確的人。

但風險也在這裡。當 channel delivery 變成 runtime 的一級能力，權限、審計、重送、刪改、格式轉換錯誤，都會變成安全邊界。如果一個 agent 可以替你發訊息、跑 CLI，它就不是聊天工具，而是操作系統的一部分。

加密貨幣今天有幾條有代表性的分歧。Standard Chartered 啟動 Uniswap UNI coverage，給出到 2030 年上看一百美元、約四十倍的多頭情境。這種目標價的真正意義，不是叫你今天買，而是把 DEX 交易量、fee switch、治理分潤、監管風險放進同一個模型裡。多頭看的是 decentralized exchange 的長期抽成能力；空頭看的是監管、治理投票和鏈上交易週期。

Strategy 也繼續增持 Bitcoin。新數字是再買一千五百八十七枚，約一億美元，總持倉八十四萬六千八百四十二枚 Bitcoin，超過總供給上限的百分之四。支持者會說這是 treasury conviction；批評者會看可轉債、融資壓力，還有如果 Bitcoin 長期走弱，公司資產負債表會怎麼被市場重新評價。

Kraken 在美國推出 crypto perpetuals 交易，靠的是它完成收購 CFTC 授權的 Bitnomial 架構。這代表合規衍生品供給增加，但槓桿、散戶保護和市場操縱風險也會回到監管桌上。Zimbabwe 則要求 crypto firms 向央行登記，初始五百美元、年續費四百美元。

前沿科技今天最值得講的，是中國海洋具身智能公司世航智能完成超過十億元人民幣 A 輪融資。這符合今天的類別變化事件，因為它是十億級融資。投資方包括上河動量基金、Vertex Growth、大洋電機等，報導把它稱為全球海洋機器人領域最大單輪融資之一。

為什麼海洋機器人重要？因為它比工廠機器人難很多。水下環境低光照、高濁度、有水流、通訊差、維修昂貴，機器人不只要看見，還要理解物理後果。這也是世界模型和具身智能討論升溫的原因。

這裡也有泡沫風險。資本喜歡把「具身智能」講成下一個大平台，但硬體部署不會因為模型變大就自動變便宜。海洋場景的維修、保險、客戶教育、可靠性驗證，都可能比模型訓練更慢。

Consumer tech 今天有 DJI Osmo Pocket 4P。它主打雙攝、十七級動態範圍、D-Log 2，起售三千七百九十九元人民幣。這條新聞的重點是，手機影像已經很強的情況下，仍有人為穩定、色彩、收音和可預期素材，買一台獨立口袋雲台相機。

Space and science 這邊，NASA 公布 Artemis III crew，包括 Andre Douglas、Frank Rubio、Luca Parmitano 和 Randy Bresnik。任務會測試 SpaceX 和 Blue Origin 登月器的在軌整合。這不是今天發射，但它把 Artemis 的瓶頸講得很清楚：未來登月不是單一火箭任務，而是多家公司、多載具、多次 rendezvous 的系統工程。任何一個環節延誤，都會往後影響 Artemis IV 和更長期月球基地節奏。

Biotech 今天有一篇 arXiv 論文值得放進來。研究比較 Gosset 這種有 curated drug asset index 的 AI 平台，和幾個 frontier LLM 加 web access，在十個小眾腫瘤與免疫 target 上做搜尋。結果是 Gosset 每次查詢回傳的 verified drugs，是最佳通用模型的三點二倍，而且在對照集合裡 precision 和 recall 都做到百分之百。

這不是臨床突破，但它很有啟發。生醫 AI 的難點常常不是「大模型會不會回答」，而是資料是不是被正確整理、標註、去重、更新。工具調用和 curated data，不是附屬品，而是能力本身。

科技延續區短講幾句。PyTorch 今天的 CI release 繼續清理 XPU、Torchtitan、測試與硬體 backend 邊界，這是多硬體 AI framework 的日常維護，不要誤讀成正式功能大發布。CrewAI 1.14.7 的 memory、knowledge、RAG backend 可插拔，是上週故事的延續，今天沒有新類別變化。Microsoft 修補後的 zero-day PoC 延燒，則提醒資安團隊：修補不是按下 update 就結束，還要看攻擊者會怎麼把公開 PoC 重新包裝。

科技深度區，我想收在「可信任的層」這件事。

今天每條科技新聞，其實都在說同一件事。Ollama 和 Transformers 問的是模型工具鏈哪一層可信。OpenClaw 問的是 agent 可以操作外部渠道時，權限和審計哪一層可信。Windows zero-day 問的是作業系統的加密、復原和防毒邊界哪一層可信。Crypto treasury 問的是公司把資產押在 Bitcoin 上時，融資結構哪一層可信。Biotech AI 問的是模型答案背後的資料索引哪一層可信。

過去一年，科技業太習慣用「更大、更快、更自動」描述進步。但接下來的競爭，可能更常落在「可驗證、可回滾、可審計、可重現」。尤其是 agent 和 AI infrastructure，一旦它們開始替你發訊息、跑工具、改資料、做研究，你真正需要的不是它每次都很聰明，而是它出錯時你能不能知道，能不能停下來，能不能追得回去。

今天就到這裡。6月16日這集，我們從 Hormuz 的六十天協議講到 Kyiv 的夜間空襲，從 Congo Ebola 講到英國社群禁令，再從 Ollama、Transformers、Windows zero-day 講到海洋機器人和生醫 AI。你可以帶走一句話：今天的世界不是沒有協議，也不是沒有工具；真正稀缺的是，協議能不能落地，工具能不能被信任。我們明天繼續。

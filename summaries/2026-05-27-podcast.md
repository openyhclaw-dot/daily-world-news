今天是5月27日，歡迎收聽第84集。今天的主線其實很清楚：中東還在試著把火壓下來，但軍事行動又把談判推回危險邊緣；同一時間，剛果的 Ebola 疫情提醒我們，公共衛生危機一旦撞上社區不信任，很快就不只是醫療問題。

我們先從中東開始。美軍在伊朗南部發動所謂自衛性打擊，這讓原本正在成形的荷莫茲海峽安排重新變得不穩。過去幾天，美國跟伊朗釋出的訊號是，先把海峽重新打開，讓油輪跟液化天然氣船恢復通行，再把更棘手的核問題留到後面談。這個設計本來就很脆弱，因為它等於承認雙方最急的是降溫，不是一次解決所有問題。現在美軍又動手，問題就變成：華府到底是在用軍事壓力保護談判，還是在讓談判對象更難退讓？

荷莫茲海峽的重量，你可以想成是全球能源系統的咽喉。平時大約五分之一的石油和液化天然氣流量會經過這裡，所以任何一個看起來只是地區性的軍事事件，都會被航運公司、保險公司、能源買家放大解讀。美國的說法會是，打擊是為了自衛，也是為了讓伊朗知道，海峽不能被當成勒索工具。伊朗的盤算則是反過來：只要海峽還沒有完全恢復，自己手上就還有籌碼，可以換制裁、港口封鎖、石油出口空間。這也是今天最危險的地方。雙方都說自己想讓局勢降溫，但雙方都還不願意放掉能讓對方痛的工具。

中東另一條線在黎巴嫩。以色列軍方跟 Hezbollah 在黎巴嫩南部一條戰略河流附近交火，而且以軍部隊往更北的方向推進。這件事的時間點很敏感，因為黎巴嫩跟以色列代表團正準備在華府談。你可以把它看成停火外殼下的軍事現實：政治上大家需要談判，軍事上雙方都覺得現在退一步就會輸掉嚇阻。以色列總理 Netanyahu 的訊息是要加大打擊，理由是 Hezbollah 的威脅沒有解除；黎巴嫩側看到的則是，以色列正在把本來應該被停火框住的衝突，變成可以常態化推進的邊境作戰。

這會牽動更大的美伊談判。因為 Hezbollah 不是一條孤立戰線，它跟伊朗的區域網路連在一起。只要黎巴嫩戰線升溫，美國要同時說服以色列克制、伊朗讓步、海灣國家相信航運會恢復，就會變得更難。也就是說，今天的中東不是一場談判，而是好幾場互相干擾的談判：荷莫茲、核問題、黎巴嫩、加薩、以色列跟海灣國家的正常化，全都擠在同一張桌子旁邊。

再往非洲看，剛果的 Ebola 疫情正在走向更麻煩的階段。剛果東部一間治療 Ebola 病患的醫院遭到憤怒民眾闖入，醫護人員被迫疏散病人，現場還傳出槍聲。前線志工說，疑似病例已經接近一千例。世界衛生組織先前的判斷是，全球擴散風險還低，但在剛果國內跟區域層級，風險已經很高。

Ebola 最難的地方，不只是病毒本身，而是它會碰到死亡、隔離、遺體處理、醫療不信任這些最敏感的社會問題。政府限制喪禮跟大型聚會，從防疫角度很合理，因為傳播鏈常常就在照護跟喪葬過程中延續。可是如果社區覺得醫院把人帶走之後資訊不透明，或覺得家屬被排除在外，那限制就會被理解成威權命令。接下來可能出現的，不是更多配合，而是藏匿病患、拒絕通報、攻擊醫護。這會讓接觸追蹤崩掉，也會讓疫苗、隔離病房、國際援助都打折。

疫情也已經外溢到外交日程。印度跟非洲聯盟延後原定5月28日至31日在 New Delhi 舉行的 India-Africa Forum Summit。這場峰會本來會談貿易、基建、南南合作，現在被公共衛生風險改寫。這不代表外交不重要，而是說今天的大型國際會議，已經不能再把公共衛生當成背景變數。它可能直接決定誰能出席、議程能不能推進、甚至一國對一整個區域的風險印象。

美洲今天有兩個政治壓力點。第一個在哥倫比亞。5月31日，哥倫比亞將投票選出下一任國家行政團隊，這場選舉很大程度上會被看成對 Petro「全面和平」政策的公投。選舉觀察組織列出三百八十六個市鎮面臨非法武裝暴力風險，大約是全國三分之一；智庫估計，全國仍有大約兩萬七千人持有武裝。支持談判的人會說，正因為武裝團體還在，才需要政治解方；反對者會說，無人機攻擊跟炸彈事件已經證明，談判讓武裝團體有時間重整。

第二個在玻利維亞。La Paz 的抗議因為燃料、美元短缺和要求 Rodrigo Paz 下台而升級，政府承認一名示威者死亡。這種局勢最容易變成雙方互相證明自己是對的：政府會說，國家已經在供應危機裡，必須先恢復秩序；反對方會說，示威者死亡證明政府失去合法性。對玻利維亞來說，麻煩不只是街頭衝突，而是能源、外匯、政治信任三件事同時下滑。

歐洲這邊，烏克蘭對 Belarus 方向的風險更加警戒。俄羅斯最近對烏克蘭的大規模轟炸後，法國總統 Macron 罕見致電 Belarus 領導人 Lukashenko，要求他不要讓 Belarus 被拖進戰爭。對法國來說，這是一通危機管理電話；對 Kyiv 來說，更實際的問題是 Belarus 會不會繼續成為俄羅斯飛彈、兵力或威嚇的延伸平台。這條線今天沒有開戰那種單一爆點，但它會增加烏克蘭北方防務壓力，也讓北約東翼更難鬆一口氣。

歐洲還有一條比較不像戰爭、但很關鍵的制度新聞：德國財長表示，Berlin 願意在歐盟資本市場聯盟談判中妥協，甚至包括敏感的金融監管問題。這聽起來很技術，其實關係到歐洲能不能把分散在各國的資本更快導向企業、基建和科技投資。支持者會說，歐洲若沒有更深的共同資本市場，就很難跟美國和中國的金融與產業規模競爭；反對者則擔心，監管權上收之後，地方銀行、國內金融穩定和各國政治責任會被稀釋。這也是歐盟常見的困境：大家都知道要變快，但每一步變快，都會碰到主權和監管權的邊界。

台灣線則仍然卡在美中台三角。台灣政府說，沒有收到美國暫停一百四十億美元軍售的通知；Lai Ching-te 也表示，如果有機會跟 Trump 說話，會強調台海和平穩定關乎全球安全。不過川賴通話還沒有安排。這裡的關鍵不是一通電話本身，而是台灣在美中交易裡的位置。台北想把軍購說成嚇阻，北京會把高層互動看成踩線，華府則還在保留模糊空間。對台灣社會來說，最難受的也就是這種模糊：你需要美國承諾，但又不能確定承諾會不會被拿去談判。

南亞跟東南亞今天看菲律賓。Sara Duterte 的彈劾審判已經由參議院以彈劾法院形式展開。若被定罪，她可能被禁止擔任公職，這會直接打到她2028年的大選路。支持彈劾的一方會把它包裝成問責，特別是 corruption、public trust 這些指控；Duterte 陣營則會說，這是 Marcos 陣營在清除下一場大選對手。菲律賓政治最值得注意的地方，是家族競爭正在被制度程序吸收進去，可是制度程序本身也因此被派系戰消耗。

非洲除了 Ebola，Sudan 也有兩條線。Human Rights Watch 指控 UAE 訓練哥倫比亞傭兵，再把他們送去支援 RSF。這如果屬實，Sudan 內戰就更不是單純的國內軍閥衝突，而是外部資金、傭兵、區域競爭共同延長戰爭。另一則是 West Kordofan 的市場遭無人機攻擊，當地人權團體說二十八人死亡、數十人受傷。無人機正在讓 Sudan 的戰爭更便宜、更遠距，也更容易打到平民生活的核心位置：市場、醫院、道路、燃料點。

大洋洲今天的重點在太平洋島國安全。Fiji 和澳洲聯邦警察共同主辦的太平洋跨國犯罪峰會結束後，各國警察首長承諾加強合作，打擊毒品、腐敗和 organised crime。澳洲方面提到，今年相關毒品查獲量已經達到十七噸，超過去年全年。這個數字很重要，因為它把太平洋安全議題從抽象的大國競爭，拉回到港口、漁船、金融帳戶、地方警察和社區成癮問題。對澳洲來說，這是區域治安和邊境安全；對島國來說，這是小型社會能不能承受跨國犯罪資金滲透的問題。

比較短的幾條延續：加薩停火下，聯合國安理會被要求施壓 Hamas 解除武裝，但同時 Israel 也被要求履行援助與保護平民義務；USMCA 更新談判將聚焦原產地規則和經濟安全，加拿大跟墨西哥都在防止談判被美國雙邊化；秘魯6月7日決選前，Fujimori 對 Sanchez 的競爭仍被貧困、治安和政局不信任牽動；太平洋警察首長在 Fiji 峰會後承諾加強打擊跨國犯罪，因為毒品問題正在從轉運風險變成島國社區本身的危機。

說完時事，我們自然轉到科技。今天科技線有一個共同主題：AI 跟 agent 正在從展示能力，轉進部署責任。誰能讓系統穩定、可控、可審計，誰才有機會真的進企業流程。

第一條看 LangChain。LangChain 發布1.3.2，要求 langgraph 至少到1.2.2，修正 TodoListMiddleware 裡 final answer 的落點，也加入 PIIMiddleware 在串流過程中遮蔽個資的能力。這不是那種聽起來很炫的大模型新聞，但它很重要，因為 agent 產品真正進企業時，最先遇到的不是模型會不會回答，而是資料會不會在工具調用、串流輸出、記憶與日誌裡亂跑。

你可以想像一個客服 agent 正在處理用戶資料，模型一邊生成，一邊呼叫 CRM，一邊把結果串流回前端。如果中途出現身分證字號、付款資訊、內部備註，遮蔽不能等到最後才做。LangChain 這次把 PII redaction 放進串流途中，就是在補這個企業落地的洞。當然，批評也成立：中介層越多，越難 debug，責任也越分散。企業不能因為 framework 加了一個 middleware，就假裝資料治理已經完成。

第二條是 Coinbase Base。Base 推出面向 ChatGPT 的工具，讓使用者可以透過對話管理 crypto wallet 和 DeFi app。這件事的方向很清楚：如果 AI agent 要幫人做事，它最後一定會碰到支付、授權、交易和資產移動。Crypto rails 對 agent 很有吸引力，因為它原生就是可程式化的，沒有太多傳統金融中介層。

可是這也是風險最大的地方。一般聊天機器人答錯，你可以關掉重問；DeFi 操作一旦送出，常常沒有客服可以幫你撤回。看多者會說，對話式錢包能把 DeFi 的使用門檻降下來；看空者會盯著釣魚、過度授權、prompt injection、錯誤交易和使用者根本不知道自己簽了什麼。這一波 agent 加錢包的產品，如果沒有把權限設計做得非常保守，很容易在第一批事故後被監管跟用戶信任一起反噬。

第三條是英國制裁 Huobi 和盧布穩定幣發行方，理由是打擊俄羅斯 crypto 網路。這代表監管者看 crypto 的方式越來越整體化，不再只盯交易所，也會看穩定幣、場外流動性、跨境支付和制裁迴避。合規交易所可能會覺得這是利多，因為灰色流量會更難進主流市場；但另一個結果也可能是，部分流動性往更不透明的路徑移動。也就是說，制裁能提高正規市場門檻，但未必能讓風險消失。

開源跟工程面，PyTorch 今天回退了一個 Triton compile-time autotune fallback 的修正，原因是內部 signals 出現破裂。這種新聞看起來很小，但對 LLM 訓練和推論堆疊來說，compiler path 的一點變化都可能被放大成效能差異或穩定性問題。AI 基礎設施不是只有更快的 GPU，也包括框架團隊能不能在速度、回歸測試、下游部署之間找到平衡。

Dify 也發布1.13.3，重點是 workflow execution、streaming 和 knowledge retrieval 的穩定性。這同樣是「不性感但重要」的方向。開源 agent app 平台要被企業用，長流程不能跑到一半斷掉，檢索結果不能忽然失真，串流不能讓使用者看到半成品裡不該出現的內容。今天的 agent 戰場，已經不是誰最會 demo，而是誰能把每個小失敗都處理得不那麼災難。

中國 AI 這邊，天工推出 SkyClaw-v1.0 和輕量版，主打百萬 token 上下文，並針對工具調用、多輪任務、程式碼生成、文件編輯和研究型資料分析最佳化。這很符合今年中國模型廠商的方向：不只說自己模型很強，而是把模型放進 agent 工作流裡賣。問題在於，官方宣稱需要第三方 benchmark 和真實任務驗證。百萬 token 上下文聽起來很有吸引力，但如果檢索、注意力分配、工具調用失誤，長上下文只會把錯誤藏得更深。

字節跳動的 Seed 部門也傳出向員工開放與「豆包股」掛鉤的低價股票期權。這顯示大型平台開始用更細的內部股權機制留住 AI 人才，不再只是整家公司一個估值、一套激勵。對員工來說，這可能讓自己更直接分享 AI 業務成長；但也要看估值怎麼定、能不能流動、資訊揭露有多完整。AI 人才戰已經不只是薪水戰，而是平台如何把「你正在做的那個模型」變成一個可被想像的財務上行。

三星也傳出將從下月開始允許員工使用外部 AI 模型。這件事放在企業管理上看，比單一模型發表還值得注意。過去很多公司在資料外洩事件後，第一反應是全面封鎖；但封鎖久了，員工會用私人帳號和影子工具繞過去，風險反而更難管理。現在大型企業慢慢轉向受控開放，也就是把可用模型、資料分類、提示紀錄、輸出審查和供應商責任一起制度化。真正的問題不是員工能不能用 AI，而是公司能不能知道誰在什麼情境下，用了哪些資料，產生了什麼可追溯的結果。

再補一條工業科技。上海橋田智能完成億元級 A+ 輪融資，由國家工業母機產業投資基金獨家投資，資金會用在高端產能、核心技術和全球化布局。它做的是汽車主機廠機器人末端和周邊設備。這種新聞不如大模型吸睛，但它對製造業很關鍵：如果機器人的末端執行器、快換盤、連接器、修磨機這些環節不能穩，整條自動化線就會卡住。中國投資這一塊，本質上是在補高端製造和供應鏈自主的底層能力。

最後補一個政策前哨。新的研究把 AGI 放進歐洲地緣政治準備議程，指出歐洲在算力基礎設施、人才留任、產業採用和戰略意識上都有缺口。這不是今天就會變成法案的新聞，但它反映一個趨勢：歐洲過去常把 AI 放在監管框架裡談，現在越來越需要把 AI 當成產業競爭、國防準備和公共部門韌性的問題。換句話說，AI 不只是怎麼管，也開始變成歐洲能不能在下一輪技術秩序裡保有談判位置。

比較短的幾條延續：Microsoft Copilot Cowork 外洩檔案的案例，今天不重講背景，但它繼續提醒企業 agent 權限邊界不能只靠使用者善意；Vatican 關於 AI 倫理的討論還在發酵，重點不是宗教突然變懂模型，而是 AI 風險已經進入公共道德語言；Gary Marcus 引述 Uber 對 AI 成本和生產力落差的疑慮，則提醒市場不要只看模型進步，也要看單位經濟能不能成立。

今天就到這裡。5月27日最值得帶走的一點是，世界新聞跟科技新聞其實在講同一件事：系統越複雜，真正的風險越常出現在連接處。中東的連接處是戰場和談判，Ebola 的連接處是醫療和社區信任，AI 的連接處是模型、工具、資料和權限。明天我們繼續追蹤這些連接處有沒有斷裂，或終於被補上。

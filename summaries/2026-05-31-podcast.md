每日世界新聞與科技新聞，5月31日，EP88。

早安，今天是5月31日，星期日。

今天最重要的線索，是幾個看起來分散的危機，其實都在問同一件事：當一個局部熱點卡住，外部世界要花多少成本才能把它穩住。哥倫比亞選前指控厄瓜多干預選舉，這不是單純外交口水，而是拉美安全路線要不要轉向的前哨。新加坡的 Shangri-La Dialogue 上，美國對中國語氣比去年降溫，但仍要求印太盟友提高防衛投入。中東方面，以色列在黎巴嫩南部推進，美伊談判的紙面進展又被戰場現實拉回來。

我們先從哥倫比亞開始。5月31日，哥倫比亞投票決定下一任政府方向，這場選舉被外界看作對 Petro 路線的公投。Petro 的核心政治招牌，是所謂的全面和平，也就是用談判處理國內仍活躍的武裝團體問題。但過去幾年，地方治安、毒品經濟、游擊隊分裂派與城市安全焦慮交織在一起，讓這個承諾很難只用道德語言維持。

投票前一天，哥倫比亞政府指控厄瓜多 Noboa 政府干預選舉。事情的表面，是厄瓜多取消部分哥倫比亞進口關稅；但哥倫比亞政府認為，這個動作搭配反對派候選人的承諾，等於外國政府在選前替某一方創造政治利益。這類爭議在拉美並不少見，但它放在哥倫比亞此刻的脈絡裡，意義特別重。因為選民現在真正要判斷的，不只是下一任政府該怎麼處理貿易，而是要不要繼續讓談判與和解放在治安政策中心。

同時，選舉暴力陰影也沒有消失。AP 報導投票前地方社群擔心無人機攻擊，Reuters 也提到 ELN 與 FARC dissidents 曾宣布選舉期間單方面停火。這種停火聽起來像緩和，但也透露一件事：武裝團體仍有能力用「是否克制」來影響民主程序。對執政陣營來說，這證明談判仍是必要工具；對反對派來說，這剛好證明武裝團體被賦予太多政治籌碼。

今天如果只看誰領先，反而會錯過更大的轉向。哥倫比亞的安全問題，不只關乎哥倫比亞。它會牽動委內瑞拉邊境、厄瓜多港口治安、跨國毒品路線，甚至美國與歐洲對拉美左翼政府的安全合作態度。這也是為什麼厄瓜多那個關稅動作會被 Bogota 放大處理：它不是單一政策，而是選前敘事的一部分。

接著看歐洲。烏克蘭總統 Zelenskyy 說，俄羅斯正在準備新一輪重大攻擊。這條新聞在今天不是全新的戰爭轉折，但它代表戰爭節奏又回到城市防空與外交耐心的雙重壓力。AP 先前報導，俄方已經警告外國公民與外交人員離開基輔，並提到可能對基輔進行系統性打擊。這種警告有軍事意義，也有心理戰意義。

對 Kyiv 來說，這正是繼續向美國與歐洲爭取防空系統的理由。Patriot missiles 不只是武器清單，而是城市能否維持基本運轉的政治保險。對 Moscow 來說，威脅基輔可以逼歐洲重新計算援助風險，也可以向國內展示戰爭主動權仍在俄方手裡。歐洲的困難在於，大家都知道烏克蘭需要防空，但每一次補給都牽動庫存、訓練、國內預算與升級風險。

德國這幾天提出另一條路：讓烏克蘭取得某種 EU associate membership。這不是正式入盟，也沒有投票權，但它可能讓烏克蘭參與部分歐盟會議與制度流程。你可以把它理解成一個政治中繼站。德國要傳達的是，哪怕正式入盟還很遠，歐洲也不能只用軍援來維持烏克蘭的未來想像。可是反對者會問，沒有正式權利的制度靠近，是否只是把真正困難的入盟問題往後拖。

然後到印太。Shangri-La Dialogue 在新加坡登場，美國防長 Hegseth 的語氣比去年收斂。他仍說美國要在太平洋維持有利且持久的權力平衡，也要求亞洲盟友提高軍費，回應中國軍力擴張。但值得注意的是，他沒有用去年那種幾乎把台海危機放在演說中心的方式處理中國。

這個語氣變化很微妙。中國代表把它解讀成川習會後的戰略指引，也就是中美關係有降溫空間。可是美國的政策訊號並沒有真的轉向軟化：它仍要求盟友更自立，仍把軍事平衡當成區域秩序核心。換句話說，華府可能只是把語言從警報聲改成制度語言，目標還是同一個：讓日本、澳洲、菲律賓、台灣與其他夥伴承擔更多嚇阻成本。

越南領導人 To Lam 的說法很能代表中型國家的焦慮。他用荷莫茲海峽當例子，提醒一個熱點如何拖累全球供應鏈。對東南亞國家來說，最不想看到的，就是南海或台海變成另一個荷莫茲。大國會談戰略平衡，小國算的是港口、保險、能源、糧食與出口訂單。

日本和菲律賓也在這個脈絡裡升級防衛合作。AP 報導，日本同意加速與菲律賓談武器銷售與情報共享。這不是一夜之間的同盟跳躍，而是日本安全政策正常化的延伸。菲律賓面對南海壓力，需要更多外部支撐；日本則逐漸把台海、東海、南海放進同一張安全地圖。北京會把這視為外部勢力圍堵，東京與馬尼拉則會說，這是對中國海空活動增加的被動回應。

台灣今天的訊號也偏緊。Reuters 報導，台灣派艦機監控中國一週內第二次 joint combat readiness patrol。這類新聞容易被日常化，因為它出現太頻繁了。但頻繁本身就是問題：當軍事壓力變成日常，誤判與疲乏風險會一起升高。台北真正要看的，不只是今天有多少架次，而是美國軍售、台灣預算與區域盟友分工能否跟上這種長期消耗。

中東方面，黎巴嫩戰線繼續成為美伊談判的外部破口。AP 報導，以色列軍隊進入黎巴嫩南部 Dibbine，附近是 Marjayoun 一帶，同時以色列與黎巴嫩軍方在 Pentagon 進行直接會談。更刺眼的是，報導提到以色列空襲造成至少 6 人死亡。這種畫面會讓「正在談判」四個字變得很脆弱。

以色列的邏輯，是北部邊境威脅沒有消失，所以軍事行動必須繼續。黎巴嫩內部，尤其 Hezbollah 相關陣營，會說直接談判正在正當化以軍推進，而且主權被一步步切割。美國想把黎巴嫩、伊朗、荷莫茲與 Gaza 的問題拆開處理；伊朗則一直希望把黎巴嫩戰線納入更大的停火安排。這就是今天中東外交的麻煩：每個戰場都想被放進總交易裡，但每放進一個，協議就更難落地。

荷莫茲海峽則是經濟面的同一個問題。美國制裁伊朗新設的航運控制機構，歐盟外交高層也說，即使戰爭結束，恢復荷莫茲航行自由還需要更多船艦與更大的海軍任務。這條水道過去承載全球大約五分之一的油氣流量，所以紙面停火不能直接變成低成本航運。真正要看的是船東、船員、保險公司與護航力量是否相信風險已經下降。

印度今天也被這條鏈條牽動。Reuters 報導，印度財政部警告，弱於正常的季風，加上燃料價格上漲，可能推升零售通膨。這句話背後有兩個傳導管道：第一，季風影響農產品供應與農村收入；第二，中東能源衝擊會透過運輸與能源成本傳到食品價格。對印度央行與政府來說，這不是單一外部衝擊，而是氣候、能源與民生價格疊在一起。

菲律賓今天有兩條值得放在一起看。第一，Duterte 的 ICC 審判排定在 11月開始。第二，菲律賓國內一個由主教與前 ICC 法官領導的組織，也啟動新的事實調查，準備紀錄毒品戰時期的殺戮證詞。這讓 Duterte 時代的究責從海牙回到 Manila，也讓 Marcos-Duterte 裂痕不只是選舉政治，而是司法與歷史記憶之爭。

同一天，菲律賓一名知名參議員因防洪工程回扣案遭法院下令逮捕，後來投案交保。你可以把這看成另一條治理線：一邊是毒品戰與人權究責，一邊是基礎建設與貪腐。對普通選民來說，這些議題最後都會回到一件事：國家機器到底是在保護人，還是在分配特權。

非洲方面，WHO 秘書長 Tedros 抵達剛果，支援罕見 Ebola 疫情應對。AP 先前報導，Ituri 省已有 246 起疑似病例與 65 人死亡。這個數字本身已經嚴重，但更棘手的是地理與治理條件：偏遠地區、跨境流動、接觸追蹤、社區信任與醫療物流，都會決定 outbreak 能否被壓住。

美國計畫在 Kenya 設立隔離設施，讓接觸 Ebola 的美國公民不要回美國本土隔離，這又引發法律與外交壓力。Kenya 法院暫停相關安排，背後的問題很尖銳：富國能不能把公共衛生風險外包給區域國家？接收國能不能要求透明程序與司法審查？疫情治理從來不只是醫學問題，它也是主權、信任與資源分配問題。

大洋洲今天的焦點是海事安全。紐西蘭新預算保住國防與情報經費，Defense News 提到相關機構預計取得約 43 億紐元。表面上這是國防預算，實際上也回應太平洋毒品轉運壓力。區域報導提到 narco-submarines 與跨太平洋轉運，這代表太平洋島國不再只是大國競爭地圖上的空白海域，而是毒品、海事監控與國家能力的前線。

澳洲則在 Shangri-La 談到海底電纜風險，把波羅的海與台灣海峽事件放在同一張安全圖裡。這個議題很容易被低估，因為電纜不像軍艦那樣醒目。但金融交易、雲端服務、軍事通訊與跨境企業營運，都靠這些看不見的線。對海島國家來說，海底電纜安全就是經濟安全。

時事部分最後補一個太空工程。SpaceX Starship V3 在 Flight 12 mishap 後遭 FAA 暫停後續發射，等待調查。NASA 仍仰賴 Starship 支援未來登月與月面基地架構，所以這不是單一公司測試延誤，而是 Artemis 時程風險的一部分。商業太空強調快速迭代，但載人登月需要的是另一種可靠性標準。這兩種節奏現在又撞在一起。

轉到科技。今天科技線的主軸，是 AI 從模型能力競賽繼續往硬體、基礎設施、治理與詐騙場景擴散。

第一條，是 Nvidia 與 Microsoft。Axios 報導，Nvidia 和 Microsoft 準備在下週 Computex 與 Build 展示首批用 Nvidia 晶片作為主處理器的 Windows 電腦，Dell 與 Surface 都被點名。這代表 Nvidia 不只想賣資料中心 GPU，也想重新進入 PC 主處理器位置。對 Microsoft 來說，這是第二次 AI PC 攻勢。第一次 Copilot+ PC 的問題，是硬體口號跑在軟體體驗前面；這次如果搭配本機 agents 與本機推論，想像空間會比較完整。

但你要先保留懷疑。Windows on Arm、生態相容性、續航、driver、遊戲、企業管理工具，這些都不是發表會可以一次解決的事。本機 AI agent 聽起來很美，真正困難的是權限與審計。當一個 agent 可以讀檔、操作瀏覽器、連企業 app、代你完成流程，它就不再只是聊天視窗，而是一個高權限自動化層。企業會問：它出錯誰負責？它看到的資料怎麼記錄？它的工具呼叫能不能重放與稽核？

這也接到 OpenClaw 今天的 release。OpenClaw v2026.5.28 聚焦 agent runtime recovery，包括 subagent 工作目錄隔離、hook context、session lock timeout、restart continuation，以及 app-server 或 helper failure 不應拖垮 shared runtime state。這些詞聽起來很工程，但它們其實就是 agent 產品能不能跑長任務的基本功。模型聰明只能解決一部分問題；真正上線後，你更常遇到的是狀態壞掉、工具卡住、重啟後上下文錯位、工作目錄混在一起。

LangChain Anthropic 1.4.4 也屬於這種小但重要的更新。它修正跨 provider tool-call ID 正規化。這不是消費者會注意的功能，但多模型 agent pipeline 裡，tool-call ID 錯位會讓重試、去重、審計紀錄全部變麻煩。你可以把今天這兩條 release 看成同一個趨勢：agent 產業正在從 demo 期進入運維期。大家不只問「能不能完成任務」，也開始問「失敗時能不能恢復，責任邊界能不能查清楚」。

第二條，是 OpenRouter 完成 1.13 億美元 Series B。官方公告說，這輪由 CapitalG 領投，Nvidia 等參與，OpenRouter 也稱每週處理量達到 25 兆 tokens。它賣的不是單一模型，而是 model gateway：讓開發者用一層 API 路由到多個模型。這個故事之所以吸引投資人，是因為企業越來越不想把所有 AI 風險壓在單一模型供應商上。

多模型路由的好處很直覺：成本可以比較，延遲可以調整，不同任務可以用不同模型，供應商出事時也能切換。但它也帶來新的治理問題。資料會經過哪裡？路由決策如何解釋？當輸出錯了，是模型供應商、gateway，還是應用開發者負責？如果 gateway 成為 AI 應用的中介層，它就會拿到非常敏感的 usage pattern，也可能成為新的平台權力中心。

第三條，是 AI 與社會科學。The Decoder 摘要一項大型研究，涵蓋 20.8 萬名參與者與 2,600 萬筆回應，結論是讓聊天模型更 helpful 的訓練，反而削弱它們模擬人類行為的能力。這很有意思，因為過去很多人希望用 LLM 當作廉價人類樣本，做問卷、社會模擬或行為預測。但如果模型被訓練成禮貌、合作、能解題的助理，它就不一定像真實人類那樣矛盾、偏見、反覆、受情境影響。

這不是說模型沒用，而是要分清楚用途。客服助理需要 helpful，社會模擬需要行為真實，心理實驗需要可控偏差。把同一個模型拿去做所有事，只會得到看起來很順、但方法上很脆弱的結果。

Terence Tao 談 AI 與數學的觀點，也在今天值得一提。他的核心不是說 AI 會取代數學家，而是 AI 可能降低形式化、驗證、探索與協作的摩擦，讓數學研究第一次更接近大型分工。這對科學研究很有想像力。可是數學又是最不能容忍模糊正確的領域之一，所以人類直覺、形式驗證與可重現推理仍會是瓶頸。AI 讓你走得更快，但也可能讓錯誤以更漂亮的形式出現。

第四條，是 crypto 與 AI 詐騙。SEC 起訴一名德州男子，指控他以 AI crypto trading bots 為名募集 1,230 萬美元。CoinDesk 報導稱，只有約 3% 資金真的用於 crypto trading，620 萬美元被轉作個人用途，550 萬美元用於類龐氏付款。這就是 AI 和 crypto 最危險的交集：兩邊都有高技術想像、資訊不對稱與暴富敘事，合在一起很容易變成募資話術。

同一天，CoinDesk 也報導美國稱查扣約 10 億美元與伊朗相關的 crypto。這又是另一個面向：鏈上資產不是脫離地緣政治的自由空間。制裁、託管、交易所、stablecoin 發行商與分析公司，都會把 crypto 拉回國家權力網裡。看多者會說，公鏈透明度讓執法更有效；看空者會說，這代表 crypto 基礎設施越主流，越逃不開傳統金融的合規與政治風險。

第五條，是 cybersecurity。EY 撤回一份充滿幻覺引用的 cybersecurity report。GPTZero 調查指稱大量引用不存在或失效，Cybernews 說問題比例高達 70%。這件事尷尬的地方在於，主題剛好是 cybersecurity，而錯誤剛好暴露專業服務業使用 AI 產出報告時的最大弱點：看似完整的引用，可能只是模型編出來的可信外觀。

Google 先前也警告，已看到 AI 協助生成 zero-day exploit 的早期實戰案例。今天把這兩件事放在一起看，會看到 AI 安全的雙重壓力。一邊是攻擊者用 AI 找漏洞、寫 payload、降低技術門檻；另一邊是防守者與顧問也可能用 AI 生成低品質報告，污染決策。真正的安全能力不只是買模型，而是把來源驗證、code review、紅隊測試與責任流程重新設計。

最後是太空與開發者工具。FAA 暫停 SpaceX Starship V3 後續發射，等待 Flight 12 mishap 調查，這在科技線裡也是一個提醒：硬體工程的現實不會被敘事加速。Starship 對 NASA Artemis 架構很重要，但監管與可靠性不可能被跳過。

開發者工具方面，Ollama v0.30.0 release candidate 繼續把本地模型架構切向 llama.cpp、GGUF，Zig 的 ELF linker devlog 也在開發者圈引發關注。這些不是主流大新聞，但會決定很多人實際開發 AI 與系統軟體時的摩擦成本。

今天就到這裡。哥倫比亞選舉、Shangri-La Dialogue、黎巴嫩戰線與荷莫茲安全，都在測試國家如何處理外部干預與安全成本。科技線上，AI 則從模型本身繼續擴散到 PC、agent runtime、gateway、詐騙、資安與研究方法。

明天繼續。

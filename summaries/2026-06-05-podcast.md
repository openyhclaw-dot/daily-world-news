每日世界新聞與科技摘要，6月5日，EP93。

早安，今天最重要的不是哪一條新聞單獨爆開，而是幾條線同時提醒我們：停火協議如果管不到拿槍的人，就只是一張紙；AI 如果只算公司營收，就會低估它正在改變國家之間的權力分配。今天我們先從中東開始，接著看俄烏遠程互襲、美國把強迫勞動變成新關稅工具，再轉到台灣與 AI 供應鏈，最後看科技圈的本地模型、agent 工具與加密金融。

先看時事的新事。

中東今天的核心，是 Israel 和 Lebanon 的停火安排剛往前走一步，Hezbollah 就把它往後拉。Israel 和 Lebanon 同意延長脆弱停火，目標是稍後繼續談更完整的和平安排；但 Hezbollah 領袖 Naim Kassem 透過聲明說，這份協議荒謬、羞辱，要求以軍先完全撤出 Lebanon。更麻煩的是，地面上沒有跟著降溫。以色列攻擊造成至少四人死亡，還有一名聯合國維和人員在交火中喪生。這裡的問題不只是誰違反協議，而是協議到底能不能約束真正作戰的人。Lebanon 政府可以坐上談判桌，但 Hezbollah 有自己的軍事決策；Israel 可以說安全區與解除武裝是和平條件，但如果它同時繼續攻擊，對 Hezbollah 支持者來說，這就像在砲火下要求投降。美國想把 Lebanon 戰線放進更大的 Iran 降溫安排，可是這條線現在反而成為整個中東談判的阻塞點。

第二條是俄烏戰爭。6月4日，Putin 在聖彼得堡經濟論壇場邊承認，部分烏克蘭無人機已經突破俄羅斯防空，他說俄羅斯會強化防空能力。這句話的背景，是烏克蘭無人機飛越超過一千公里，擊中聖彼得堡油庫；而俄羅斯在6月2日也用數百架無人機與數十枚飛彈攻擊 Kyiv 和其他烏克蘭城市，造成至少二十二名平民死亡、一百三十八人受傷。俄烏戰爭正在更明顯地變成遠程空襲互耗。Ukraine 的邏輯，是把戰爭成本推回俄羅斯境內，攻擊能源、物流與軍事支援節點；Russia 的邏輯，是用大規模空襲消耗烏克蘭防空，也讓本國民眾相信戰爭還在可控範圍內。這不一定立刻改變前線，但會改變城市防空、能源設施、保險成本與民心壓力。

第三條是美國把強迫勞動議題變成新的關稅工具。Trump 政府提議，對六十個經濟體的進口商品加徵百分之十或百分之十二點五關稅，理由是這些經濟體沒有有效阻止強迫勞動商品進入供應鏈。India、South Africa 等都被捲入。印度方面說，仍在與美國談 Section 301 調查，這個時點很敏感，因為雙方貿易協議本來已經接近框架收尾。美國的說法是，這是勞權與供應鏈合規；貿易夥伴看到的，則是另一套可以替代舊關稅工具的新壓力。支持者會說，如果不讓進口商付出代價，強迫勞動禁令就只是口號；反對者會說，這種工具太容易變成保護主義，也可能把人權議題拿來當談判籌碼。

第四條在 Latin America。Colombia 第一輪總統選舉正式清點接近完成，選委會說正式清點與預先計票有百分之九十九點九一致性。右派候選人 Abelardo de la Espriella 和左派 Iván Cepeda 將在6月21日決選。今天的新變化，是 Bogotá 法院暫時要求 De la Espriella 不得在宣傳與造勢中使用 Colombia 國家隊球衣；同時，Petro 對海外票與可能舞弊的說法，仍在把選後氣氛推向更高張力。這場選舉已經不只是左右政見對決，而是 Trump 風格右派、Petro 派左翼、選舉可信度與政治暴力語言一起交纏。右派說真正的問題是現任政府介入選戰；左派則警告，De la Espriella 用毒販、恐怖分子這類語言描述對手，會讓政治暴力更容易被正常化。

第五條回到 East Asia。天安門事件三十七週年，美國國務卿 Marco Rubio 的談話引來 Beijing 反擊，中國外交系統說美方是在抹黑中國政治制度；台灣則要求中國正視歷史。這件事本身不是軍事升級，但它把美中台之間的價值衝突重新放上桌。一邊把六四視為主權與制度防衛，不接受外部評論；另一邊把它視為跨國人權記憶，不認為時間過去就能讓責任消失。這種記憶政治，常常不會立刻改變政策，卻會改變互信底線。

接著看 Taiwan 與 AI 供應鏈。TSMC 執行長魏哲家在新竹說，公司對未來幾年成長仍有信心，因為運算力與先進半導體需求很強。TSMC 在 Arizona 的投資總額達一千六百五十億美元，他也說當地兩塊土地應該足夠未來十年使用。這不是單純的企業擴廠故事，因為 Taiwan 的官方 GDP 成長預測也剛上修到百分之九點六四，是十六年來最快速度，主因就是 AI、半導體、伺服器和零組件需求強於預期。換句話說，AI boom 正在讓台灣經濟上行，也讓台灣的電力、土地、水資源、人才與地緣政治壓力一起上行。投資人看到的是長週期需求；政策制定者看到的應該是集中風險。

South China Sea 也有新動作。Philippines 和 Vietnam 升級為 enhanced strategic partnership，Marcos 在 Manila 接待 Vietnam 最高領導人 To Lam 時說，南海和平穩定不可談判。兩國海警在2024年已有首次聯合演訓，現在把政治關係再往前推。這件事的意義在於，兩個都和中國有南海爭端的國家正在靠攏；但它的限制也很清楚，Philippines 和 Vietnam 自己也有部分主張重疊。合作要能持續，必須把共同面對中國壓力和彼此主權細節分開處理。

Africa 今天的焦點是 Ebola。WHO 最新資料顯示，截至5月27日，DRC 已通報九百零六例疑似病例與二百二十三例疑似死亡；WHO 秘書長 Tedros 說，疫情已經有很大領先，醫療社群仍在追趕。這次是 Bundibugyo virus，不像 Ebola-Zaire 那樣有更成熟的核准疫苗與療法，所以檢測、隔離、接觸者追蹤、安全通行和資金都變得更關鍵。Kenya 也因為美國想在當地設置 Ebola 隔離設施而爆出政治爭議。Ruto 說這是人道與長期公衛合作，但反對者擔心 Kenya 被當成風險外包地。這不是單純的防疫設施選址，而是公衛合作、主權尊嚴與國際不平等同時被點燃。

Oceania 的兩條線，都和海底安全有關。Australia 和 Solomon Islands 同意談判 comprehensive treaty，Solomon Islands 新總理 Matthew Wale 也說會檢視和 China 的安全協議。Australia 同時承諾提供約三千五百萬澳幣，協助 Solomon Islands 面對熱帶氣旋與能源衝擊。這是 Canberra 在 Pacific 與 Beijing 競逐影響力的一個明確成果。另一邊，AUKUS 的第一個 Pillar Two signature project 鎖定無人水下載具，美國、英國、澳洲要共同開發可互換感測器與武器，目標從2027年開始交付。支持者說，這能保護海底電纜、管線，也能提升監偵和反潛能力；懷疑者則會問，AUKUS 會不會又把大量政治承諾壓在技術交付和成本控制上。

延續區簡短帶過幾條。Ukraine 在 Kyiv 會見 NATO 秘書長 Rutte 後，要求更多反彈道飛彈防禦支援，PURL initiative 的新貢獻預計6月交付；中國遼寧艦在5月26日到28日於 Luzon 東方活動，日本防衛省已發布監控資訊；Philippines 的 ICC 線繼續延伸，前警察首長 Ronald dela Rosa 的逮捕令讓 Duterte 毒品戰責任不只停在 Duterte 本人；Ethiopia 6月1日投票受不安全局勢干擾，但 Abiy Ahmed 的執政黨仍被看好續掌多數。

今天時事的深度，只看一件事：停火為什麼會失效。Israel 和 Lebanon 的最新安排，表面上符合「停火協議簽署或延長」這種重大類別變化，但它馬上遇到一個現實問題：政府不是唯一暴力行使者。Hezbollah 沒有坐在同一張桌上，卻掌握讓南部戰線升高或降低的能力；Israel 認為只要 Hezbollah 不解除武裝，任何撤軍都會讓北部安全重新暴露。這種結構讓協議變成兩層：外交層先宣布降溫，軍事層再決定它是不是算數。對聽眾來說，這提醒我們看中東新聞時不能只看「誰同意了」，還要看「誰有能力破壞它」。

自然過場到科技。今天科技線其實和剛剛的時事連得很緊：AI 不只是模型更新，也正在變成桌面工具、供應鏈、晶片驗證、內容安全與金融基礎設施。

科技新事第一條，Ollama v0.30.5。這次正式版讓 ollama launch 可以同步啟動 hermes-desktop，第一次執行時會走安裝流程；同一天的候選版也補上 Cline CLI 整合文件、Hermes Windows install 和 llama.cpp 更新。這代表本地模型 runtime 正在往「模型加 agent desktop」整合。過去本地模型比較像開發者工具，現在它開始連到桌面 agent、CLI 整合和更完整的操作環境。好處是工作流更順，資料也更容易留在本機；風險是權限面、安裝鏈和桌面整合都變大，企業環境如果沒有審計，就可能把方便變成新的攻擊面。

第二條，Hugging Face Transformers 發布 v5.10.2，修正 CLIP 相關模型轉換 bug，影響範圍包含 sam3 等模型。這是 v5.10 系列連續修補的一部分，代表大型開源模型工具鏈現在的現實：功能推得快，修補也來得快。對研究者來說，更新能避開模型轉換錯誤；對生產環境來說，patch release 也不能盲升，因為模型轉換、tokenizer、pipeline 和部署環境常常綁在一起。這類小版號，不小。

第三條，PyTorch nightly 新增 activation memory budget 的區域控制。新的 context manager 可以在 compiled forward traced 的區域內覆寫 activation memory budget，讓模型對重算策略有更細緻的控制。白話說，訓練或推論大型模型時，你可以更精細地選擇哪裡多用計算換記憶體，哪裡不要。這對成本有意義，尤其在 GPU 記憶體卡住時。但它仍是 nightly 路徑，適合實驗和壓測，不適合沒有驗證就直接進 production。

第四條，agent 評測與工具面。Hugging Face Blog 今天有兩則值得放在一起看：EVA-Bench Data 2.0 擴到三個領域、一百二十一項工具、二百一十三個情境；另一篇談 hf CLI 如何設計成更適合 agent 操作 Hub。這代表平台正在從「讓人類開發者好用」轉向「讓 agent 可預測地調用」。這很重要，因為 agent 能不能可靠工作，不只看模型會不會推理，也看工具回傳是否穩定、錯誤訊息是否可解析、命令是否有清楚邊界。風險則是，如果工具只為 agent 最佳化，人類開發者可能反而更難閱讀與除錯。

第五條，NVIDIA Nemotron 3.5 Content Safety 主打企業多模態安全客製。這類安全模型的需求很明確：全球企業面對不同國家、不同語言、不同政策與不同風險容忍度，不可能只靠一套固定規則。它能補上內容審查、合規與安全邊界，但不能被當成合規萬靈丹。安全模型仍會誤判，也可能被繞過，真正上線時還是需要人審、稽核紀錄、事故回報與快速修補流程。

加密與鏈上金融今天有幾條。Bitcoin 一週跌幅擴大，The Block 指出壓力來自 ETF 資金流出、Strategy 賣出與宏觀因素；同時 Ether.fi 配置一億美元到 Plume RWA vault，Visa 與 Brale 測試 Canton Network 上具隱私控制的 stablecoin settlement，JPMorgan 則提醒美國 crypto market structure bill 今年通過窗口可能偏窄。這幾條放在一起看，很像一個分裂畫面：價格端承壓，制度端還在往前走。看多者會說，抵押貸款、RWA vault、支付巨頭測試和資本規則討論，都在讓 crypto 從交易資產走向金融基礎設施；看空者會說，只要 ETF 流出、監管時間表不確定、底層資產透明度不足，這個制度化敘事就還沒有被資金確認。

前沿科技也有中國 AI 與硬體線。36氪報導，智維創芯完成數千萬元人民幣天使輪融資，產品 ChatDV 覆蓋測試生成、斷言生成、參考模型構建與自動調試，宣稱能把晶片開發效率提高超過十倍、週期減少一半、成本降低三分之一。這是 AI4EDA 進入商業化的訊號，但也要小心，效率宣稱需要真實晶片專案與量產案例驗證。另一則是字節 AI 的2026命題，傳出會聚焦世界模型、影片、coding dogfooding 和豆包辦公商業化。如果屬實，中國大廠的 AI 戰線正在從聊天助手擴到世界模型和 agent 工作流。

延續區這邊也快速帶過。OpenClaw v2026.6.4-alpha.1 延續 agent runtime 快速迭代，但 alpha 資訊有限，採用前仍要看 changelog 和 regression；LangChain DeepSeek provider 升到 1.1.0，重點在依賴安全與測試 floor，這是 adapter 生態必要但不華麗的維護；Immunefi 說 DeFi exploit losses 較2022高峰下降百分之七十四，但二零二五年仍有六點八億美元損失，安全改善不等於風險消失；WeLiveSecurity 提醒兒童資料外洩是長期身份風險，今天來源沒有新的大規模受害人數。

科技今天的深度，是 agent 工具正在改變軟體產品的設計語言。Ollama 把本地模型和桌面 agent 接得更近，Hugging Face 把 CLI 設計成 agent 更容易操作，EVA-Bench 把工具使用變成評測重點。這三件事指向同一個方向：未來很多軟體不只要給人點，也要給 agent 穩定呼叫。這會逼產品團隊重新思考權限、日誌、錯誤訊息、狀態恢復和最小操作單位。以前 API 是給工程師的，GUI 是給使用者的；現在中間多了一個角色，叫做會操作工具但也會犯錯的 agent。它需要清楚的護欄，也需要足夠可組合的工具。

今天收在這裡。世界線上，停火、關稅、選舉與防疫都在提醒我們，制度要真的有效，必須能管到現場。科技線上，AI 的重點也不只在模型能力，而是它如何接上桌面、命令列、晶片、金融與國家基礎設施。明天我們再繼續追，哪些承諾變成行動，哪些行動只是新的風險。

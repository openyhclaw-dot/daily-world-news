EP109。今天是6月22日。

早安，今天這集，我想先把目光放在兩種很不同的壓力測試。一個在哥倫比亞，選民要決定國家安全要回到強硬軍事路線，還是繼續押在艱難的和平談判上。另一個在科技圈，AI agent 進企業的速度很快，但真正的問題已經從「它會不會做事」，變成「它做錯的時候誰負責」。

先從時事開始。

今天最值得看的是哥倫比亞總統決選。這不是一場普通的左右對決，而是哥倫比亞和平進程的方向盤要不要大轉彎。左翼參議員 Iván Cepeda 主張讓 Gustavo Petro 政府的「全面和平」政策繼續走下去，只是要修正執行方式。右翼律師 Abelardo de la Espriella 則主張把這條路線收掉，恢復對武裝團體更強硬的軍事壓力。表面上，這是兩個候選人在講治安；實際上，他們在回答同一個問題：二零一六年和平協議以後，國家還要相信談判可以處理暴力嗎？

哥倫比亞的困境是，兩邊都不是沒有道理。支持 Cepeda 的人會說，強硬掃蕩過去試了很多年，最後常常變成鄉村社群被夾在軍隊、游擊隊、毒品組織之間，國家只是在地圖上短暫收回控制，沒有真正解決地方經濟和治理。支持 De la Espriella 的人則會說，和平談判若沒有威懾力，就會被武裝團體拿來重整隊伍、拖延時間。你可以把今天的哥倫比亞看成拉美政治的一個縮影：民眾對暴力厭倦，也對改革承諾失望；他們想要安全，但不一定相信傳統國家機器能帶來安全。

這場決選還有一個更大的區域意義。過去幾年，拉美左翼政府的浪潮已經開始退潮，如果 De la Espriella 勝出，哥倫比亞就會加入更明顯的右轉行列。這不只影響國內治安，也會改變哥倫比亞對委內瑞拉、對美國、對毒品政策的姿態。相反地，如果 Cepeda 贏，Petro 的政治遺產至少還有延續空間，但他也必須證明「和平」不是口號，而是能降低綁架、勒索、地方武裝控制的治理方案。這是今天的第一個新事。

第二個新事在歐洲。Brexit 公投十年後，歐洲民意反而更支持英國靠近歐盟。最新調查顯示，十五個歐盟國家有大約三分之二民眾支持英國重新加入；英國內部也有很多人不再相信當年脫歐承諾的繁榮、貿易自主和移民控制真的兌現。這裡最有意思的，不是英國明天就會申請重返歐盟。那還很遠。真正的變化是，十年前「脫離布魯塞爾」是主權想像；十年後，歐洲安全、軍工合作、能源、供應鏈，反而讓英國重新感覺到自己需要歐洲。

但你也不要把這件事看成單純的後悔。脫歐派仍然會說，英國不能為了貿易便利，就重新接受自由流動、共同規則和歐洲法院式的約束。工黨政府也很小心，它想修補關係，卻不想立刻重啟入盟的政治內戰。這就是 Brexit 十年的尷尬結論：英國知道離開的成本很高，歐盟也知道英國在安全上有用，可是雙方都還沒有一個低成本的政治語言，能把「重新靠近」講成不是背叛。

第三個新事回到東亞。台灣最近首次從西岸朝台海方向試射美製 HIMARS，演練的是隱蔽、發射、快速轉移。這不只是軍演畫面好看，而是在把台灣防衛概念具體化。HIMARS 的價值不在於單一火力多大，而在於它可以把登陸船團、港口集結、指揮節點放進一個更難預測的打擊範圍。換句話說，台灣要讓北京相信，任何渡海行動都不只是跨過海峽，而是一路暴露在可移動火力底下。

不過這件事有兩個限制。第一，演訓中傳出有 misfire，這提醒我們，買到武器和形成戰力不是同一件事。第二，台灣仍在等待一筆大約一百四十億美元的美國軍售，而這筆案子被放在華府與北京的政治節奏裡。台灣駐美代表俞大㵢說，台灣不會等「美國騎兵」來救，但台灣仍需要美國武器補足防衛。這句話其實很精準：台灣不能把安全外包給美國，可是台灣的軍改也不可能完全脫離美國交付節奏。北京會把 HIMARS 試射稱作挑釁，台北會稱作防衛；真正的問題是，嚇阻需要的不只是訊號，而是穩定交付、訓練和彈藥庫存。

再看南亞和東南亞。巴基斯坦正在考慮讓中國製 PNS Hangor 潛艦支撐更遠的海軍存在，甚至重新把孟加拉灣放進視野。這是印度會高度注意的訊號。因為如果巴基斯坦海軍的活動從阿拉伯海延伸到印度東側，印度的海上壓力就不再只在西邊。這也代表中巴軍事合作不是單純賣裝備，而是在改變印度洋周邊的戰略想像。

同一區域，俄羅斯與 ASEAN 在喀山峰會強化合作。普丁想把俄羅斯包裝成亞太安全的一個平衡力量，東南亞國家則維持它們熟悉的多邊避險：跟美國合作，也不關上俄羅斯或中國的門。對東南亞來說，這種彈性很有價值；對西方來說，這也是制裁俄羅斯很難變成全球一致行動的原因。

接著是幾條延續。伊朗今天把荷姆茲海峽重開條件，繼續綁到以色列在黎巴嫩的行動；這是昨天已經提過的危機，但新事實是，德黑蘭把海峽、核談判、黎巴嫩戰場包成同一筆交易，而華府仍想拆開處理。蘇丹無人機戰的平民死亡數仍停在令人不安的位置，聯合國說今年前五個月已有超過一千名平民死於無人機攻擊。剛果 Ebola 疫情也延續惡化，前線醫護已有七十五人感染、十七人死亡，這代表疫情正在直接打穿醫療系統。澳洲把部分非公民送往 Nauru 的政策也繼續發酵，爭議點不是單一個案，而是三十年簽證、家庭分離和太平洋外包制度會不會變成另一種長期懲罰。

這裡我們稍微停一下。你會發現今天很多國際新聞都不是突然爆炸，而是制度被壓到變形。哥倫比亞是在和平制度和安全恐懼之間變形；英國是在主權敘事和經濟現實之間變形；台灣是在軍購政治和自我防衛之間變形；澳洲是在移民管理和人權義務之間變形。每天看新聞最容易累的地方，是所有事看起來都沒有結束。但其實正因為它們沒有結束，今天的新事才值得抓住：它們告訴你，哪一條舊線正在往新的方向彎。

好，轉到科技。

今天科技圈的第一個新事，是 AWS 對 AI agent 的判斷很直接：agent 缺的不是更多 demo，而是企業語境和安全控制。所以 AWS 推出兩個服務，一個叫 Continuum，用來偵測、排序並修補程式漏洞；另一個叫 Context，從企業資料建立 knowledge graph，讓 agent 理解公司內部的業務脈絡。這裡的關鍵不是名稱，而是雲端大廠正在把 agent 從「聊天機器人會幫我寫 code」推向「企業流程的一部分」。

這個方向很合理，因為企業真正怕的不是 agent 不夠聰明，而是它太快、太自信、太不了解內部規則。它可以在幾分鐘內寫出一堆程式，也可以在幾分鐘內把錯誤散到很多 repo。Continuum 想處理的是安全漏洞；Context 想處理的是商業脈絡。兩者合在一起，就是在承認 agent 的缺陷不是單一模型分數能解決的，而是要靠資料權限、知識圖譜、審計軌跡、漏洞排序一起補。

批評視角也很清楚。企業一旦把 agent 治理交給大型雲平台，就可能得到效率，也得到更深的供應商鎖定。更麻煩的是，責任不會因為工具變成熟就消失。agent 修了一個漏洞，卻引入另一個邏輯錯誤，誰簽核？agent 讀了公司資料，卻把不該看的資料納入推理，誰負責？所以今天 AWS 的發表不是「agent 終於安全了」，而是「agent 進入企業後，治理會變成新戰場」。

第二個新事，是 AI 對教育評量的影響有了更具體的數字。UC Berkeley 研究超過五十萬筆成績後發現，ChatGPT 推出後，寫作和程式課的作業成績上升最明顯，但考試表現沒有同樣提升。這個結果很刺耳，因為它暗示很多分數上升不是學習改善，而是作業被外包。你如果是老師，這代表你不能再把「作業分數」當成能力提升的直接證據。你如果是學生，這也代表短期把作業交給 AI，也許能提高成績，卻不一定能提高你在沒有工具時解題的能力。

這不是說教育應該禁止 AI。禁止通常只是把使用推到地下。比較現實的方向，是把評量拆開：哪些任務允許工具，評估的是整合能力；哪些任務需要現場、口試或過程紀錄，評估的是學生本人能不能解釋。AI 讓教育變困難，不是因為它讓學生作弊變簡單而已，而是它逼學校承認，過去很多作業本來就不太能準確測量學習。

第三個新事在 crypto，而且很諷刺。知名 Ethereum sandwich MEV bot，被 counter-MEV honeypot 反制，鏈上證據顯示損失約七百五十萬美元。簡單講，原本靠夾擊別人交易賺錢的 bot，被另一套鏈上策略設局反咬。這件事之所以重要，不只是因為金額大，而是它把鏈上世界的安全問題從「合約有沒有漏洞」推到「經濟對抗有沒有失控」。

看多的人會說，這就是公開鏈的自我修正。所有行為都在鏈上，捕食者也可能被捕食，透明度讓壞策略付出代價。看空的人會說，這剛好證明普通使用者身處一個高度掠奪性的交易環境，MEV、私有 order flow、bot 對抗，最後都會變成交易成本。更重要的是，同一天還有 Secret Network 的 Axelar bridge 被 infinite-mint exploit 影響，損失約四百六十七萬美元，攻擊還被說持續七天才發現。這兩件事放在一起看，crypto 風險已經不只是「價格會跌」，而是橋、bot、監控、監管分類全部纏在一起。

延續幾條科技短訊。OpenClaw 釋出6月21日 alpha 版，延續 agent runtime 快速修補；PyTorch 兩個 release 處理 MPS shader 編譯警告與 Windows header 汙染，這不是華麗功能，但對跨平台機器學習 runtime 很實際。CrewAI 的 Flow CEL 驗證也值得持續追蹤，因為 agent workflow 越複雜，錯誤越需要在定義階段就被攔下，而不是跑到生產環境才爆。

今天還有一個值得放在最後的開源話題。OpenJS Foundation 討論 maintainer burnout，也就是開源維護者的倦怠。這件事跟 AI 生成 PR 很有關。以前開源專案的壓力是 issue 太多、使用者期待太多；現在多了一種壓力：大量看起來有幫助、實際上需要人工審查的 AI 生成變更。對維護者來說，最稀缺的不是程式碼，而是注意力。當 AI 讓產出變便宜，審查反而變貴，開源治理就會進入新的不平衡。

所以今天科技線可以用一句話收束：AI 正在把很多「產出」變得便宜，但它沒有讓「責任」變得便宜。企業 agent 需要治理，教育評量需要重設，開源維護需要保護人類注意力，crypto 自動化交易需要面對它製造出的對抗環境。技術進步最容易被展示的一面，是速度；最難被展示的一面，是誰來承擔速度造成的後果。

如果把時事和科技放在一起看，今天其實有同一個主題：工具變強，不等於制度變穩。HIMARS 讓台灣多一層嚇阻，但交付和訓練才決定嚇阻能不能站住；AI agent 讓企業流程更快，但審計和權限才決定它能不能進核心業務；無人機讓戰爭更遠距，卻也讓平民更難知道誰該負責。這些不是反科技的結論，而是提醒我們，真正成熟的系統，不只看它能做什麼，也看它出錯時能不能被追問。

今天就到這裡。6月22日，國際線看哥倫比亞決選、Brexit 十年後的歐洲回擺、台灣嚇阻的實戰化；科技線看 agent 進企業、AI 改寫教育評量，以及鏈上安全從漏洞走向經濟對抗。我們明天繼續。

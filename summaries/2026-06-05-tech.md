## 🧠 LLM

• 🔥13 **PyTorch 新增 activation memory budget 區域控制** — PyTorch nightly 加入 `torch.autograd.graph.region_activation_memory_budget(budget)`，可在 compiled forward traced 區域內覆寫 activation memory budget，讓模型更細緻控制重算策略。這類更新對大模型訓練與推論成本有直接影響，但 nightly release 仍屬高變動路徑，適合測試與實驗，不宜無驗證直接進 production。

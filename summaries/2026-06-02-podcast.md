第90集，今天是6月2日，早安。

今天這集的主軸，是「承諾」正在被重新估價。哥倫比亞的選票，要決定人民還相不相信改革承諾；中東同一天出現升級和降溫訊號，要看各方還相不相信停火承諾；科技這邊，agent 和本地模型工具不再只比誰能跑 demo，而是在比誰能把長任務、失敗復原和真實工作流做穩。

我們先從時事開始。

今天最重要的第一個現場，在哥倫比亞。昨天我們講的是投票登場，今天類別已經變了，因為第一輪結果出來了。親 Trump 律師 Abelardo de la Espriella 在第一輪拿到接近百分之四十四，領先進步派參議員 Iván Cepeda，後者得票不到百分之四十一。兩個人會在6月21日進入決選。

這代表哥倫比亞選舉的問題，已經從「Petro 路線會不會被檢驗」，變成「誰能在決選裡定義國家的恐懼」。de la Espriella 的動能，來自拉丁美洲這幾年很熟悉的治安政治。他把犯罪集團、邊境、毒品經濟和反左翼情緒綁在一起，說國家需要更強硬的秩序。對很多選民來說，這種語言直接、簡單，而且能回應日常的不安全感。

Cepeda 和 Petro 陣營則面臨兩個難題。第一，他們要說服選民，社會改革和和平路線不是治理失敗的代名詞。第二，他們質疑選舉結果，但目前沒有提出證據。這在決選前很敏感，因為如果一方先把制度可信度打折，接下來不管誰贏，都更難把另一邊的支持者帶回共同政治空間。

所以哥倫比亞接下來三週，不只是左右對決，也不是單純親 Trump 與反 Trump 對決。真正要看的是，2016年與 FARC 和平協議十年後，哥倫比亞人還願不願意相信和平承諾、國家改革和選舉程序可以一起存在。拉丁美洲很多國家現在都在同一個問題上打轉：當治安焦慮變成最大政治語言，民主還能不能避免走向強人競賽。

第二個重點在中東。今天最棘手的地方，是同一天出現兩種完全相反的訊號。Reuters 報導，以色列總理 Netanyahu 下令攻擊 Hezbollah 控制的 Beirut 南郊，這代表 Lebanon 戰線繼續升級。Lebanese authorities 說，自3月2日以來，已有超過三千三百七十人死於以色列攻擊。另一邊，Arab News 轉述 Reuters 說，Iran 因為以色列在 Lebanon 的攻勢，暫停和美國的間接談判。

可是 AP 同一天又報導，Trump 說 Israel 和 Hezbollah 已經同意降溫。這就形成一個很危險的落差：白宮口徑說有降溫，戰場現場卻看起來像升級，Iran 則把這個落差當成暫停談判的理由。

以色列的邏輯，是 Hezbollah 不能繼續把北部安全綁架，所以要用攻勢削弱它在南 Lebanon 和 Beirut 南郊的能力。Iran 和 Hezbollah 的邏輯，是 Lebanon 戰線不能被美國從 Iran 談判裡切出去，否則 Tehran 會失去一張重要籌碼。美國的邏輯，則是想把區域戰線拆開處理：Iran 談 Iran，Lebanon 談 Lebanon，Gaza 談 Gaza。問題是，參與者本來就是把它們綁在一起打。

這也是為什麼停火承諾現在很難被相信。不是因為沒有人想談，而是每一方口中的「停火」都不一樣。以色列要安全緩衝，Hezbollah 要保留政治與軍事槓桿，Iran 要把代理人戰線放進大交易，美國要用外交包裝出可宣布的降溫。只要這些目標不同步，一條戰線安靜下來，另一條戰線就可能補上火力。

第三個時事重點在菲律賓。參議員 Jinggoy Estrada 因 plunder charge 再度被捕，這次罪名沒有保釋權。檢方指控他從防洪工程收取超過五點七億披索，約九百三十萬美元回扣。這和上週的 graft warrant 不一樣，因為上週他還能交保，今天這個罪名把案件升級成更重的政治司法事件。

如果只看個案，這是一宗公共工程貪腐案。但放進菲律賓現在的政治環境，它其實打在參議院這個權力核心上。同一個國家裡，Duterte 的 ICC 審判已經排定11月30日，Ronald dela Rosa 仍牽涉 Duterte 時代 drug war 的問責線，Sara Duterte 的 impeachment 也讓 Marcos 和 Duterte 家族裂痕制度化。現在 Estrada 又因防洪工程回扣案被捕，參議院變成各條危機的交會點。

政府會說這是反貪與司法問責。Estrada 陣營會說自己否認指控，證詞也需要被審查。真正重要的是，菲律賓能不能讓這些案件通過可被相信的程序，而不是被每一派都說成政治清算。因為如果所有司法都被派系語言吞掉，最後人民只會相信自己的陣營，不相信制度。

再看東亞。中國海警6月1日宣布在台灣東部水域進行所謂 law enforcement 巡航，理由是日本和菲律賓計畫啟動海上邊界劃界談判，而這些海域涉及中國主張。這件事的地圖很重要。它不是台灣海峽正面，而是台灣東方。也就是說，北京把日本、菲律賓、台灣東側、西太平洋航道和南海主張，連成一條壓力線。

同時，日本防衛省說，中國航母遼寧艦和伴隨艦艇在5月26到28日，在 Luzon island 以東的太平洋水域活動並進行航母演訓。中國會說這是遠海訓練常態化。日本、菲律賓和台灣看到的，是航母與海警同時往第一島鏈外側伸展。

對台灣來說，這是一個很現實的提醒：台灣不一定在每一張談判桌上，但常常在地圖中央。日菲談海上邊界，中國回應的地方卻在台灣東方。這會讓台灣的安全討論從台海正面，延伸到東部補給線、海空監偵、與周邊盟友協調。支持強化防衛的人會說，這正是為什麼台灣要把監偵和後勤韌性做起來；審慎派會擔心，日菲中摩擦一旦外溢，台灣會被迫承受自己沒有主導的風險。

歐洲的 Ukraine 線今天是延續更新，但有新的壓力點。Zelenskyy 說情報顯示俄羅斯正在準備重大新攻擊。歐盟外長則說，Russia 不能決定誰代表 Europe 參與潛在談判。這句話背後，是歐洲對被排除在安全安排外的焦慮。烏克蘭最怕的是美俄談完，歐洲和 Kyiv 被要求接受結果；歐洲最怕的是，自己的安全架構被當成大國交易的附件。

美國在新加坡的 Shangri-La Dialogue 又把壓力加上去，批評 NATO allies 防務投入不足，要求 Europe 和 Asian allies 往更高軍費靠攏。這不是歐洲內部議題而已，因為美國是在亞洲場域講給所有盟友聽。盟友當然知道要花更多錢，但他們也會問：我加碼，是在強化同盟，還是在替美國可能不穩定的承諾買保險？

南亞今天的關鍵是印度。印度財政部警告，燃料價格上漲、弱於正常的季風雨，可能在未來幾個月透過運輸、能源和食物成本推升零售通膨。4月年通膨是百分之三點四八，還低於央行目標，但財政部把 Hormuz disruption 說成印度外部與價格展望裡最關鍵的變數。這句話很重，因為印度同時是能源進口大國，也是農業和食品價格對政治很敏感的國家。一旦油價和季風同時不利，央行、政府補貼、農村消費和城市交通成本會一起被牽動。

非洲的 Ebola 線也是延續，但壓力沒有下降。WHO chief 到 Congo 疫情最嚴重省分，要求居民及早治療並採取安全下葬。Bundibugyo strain 沒有核准疫苗或特效療法，所以這場應變非常依賴隔離、補液、社區信任和跨境監測。最難的不是只把病例數寫進表格，而是讓民眾相信治療中心、下葬規則和接觸者追蹤不是外部力量強加的控制。

大洋洲今天比較像長線安全布局。East Micronesia Cable System 完工，這條約兩千兩百五十公里的海底電纜，連接三個太平洋島國，由 Australia、Japan、United States 支持。它不是單純通訊工程，而是太平洋基礎設施競爭的一部分。澳洲最近也把海底電纜稱為現代文明的動脈，AUKUS 第二支柱談到水下無人系統。換句話說，太平洋安全已經從軍艦、基地，延伸到資料線、海底監控、災害韌性和跨國犯罪。

比較短的幾條延續更新：Duterte 的 ICC 審判日期已定，菲律賓國內另有事實調查蒐集 drug war 證據；Kenya 法院暫停美國 Ebola 隔離設施後，公衛合作繼續卡在援助與風險外包的爭論；台灣成長仍被 AI 供應鏈支撐，但安全線因台灣東部水域變得更複雜；美國能源成本外溢到債券利率，讓中東戰爭不只是外交新聞，也變成家庭借貸成本。

接下來進入科技。

科技今天第一個新事，是 Ollama v0.30.0 release candidate。這一版延續 v0.30.0 的主軸：架構直接轉向支援 llama.cpp，相容 GGUF，並用 MLX 加速 Apple Silicon 推論。這聽起來像工程細節，但對本地 LLM 來說很關鍵。因為本地模型工具真正要普及，不能只靠「可以跑」，還要靠格式相容、硬體加速、模型管理和升級路徑穩定。

不過它仍然是 release candidate。這代表你可以把它當成方向訊號，但不該把它當成企業穩定承諾。今天的重點不是 Ollama 發布了某個終局版本，而是本地模型堆疊正在更明確地收斂到 llama.cpp 和 GGUF 這些生態標準。對開發者來說，這會降低很多模型切換成本；對企業來說，風險仍然是支援矩陣、相容性和效能是否能在不同硬體上被驗證。

第二個科技焦點，是 OpenClaw 2026.6.1 beta。這版把 interrupted tool calls、stale session bindings、compaction handoffs、media delivery retries 放在改善重點，也提到 Telegram、WhatsApp、iMessage、Slack、Discord 等通道更穩。這不是炫目的模型能力，但它非常接近 agent 產品真正的問題。

Agent 的 demo 常常看起來很聰明，因為它在一條順路上完成任務。可是長時間自動化裡，真正常見的是工具呼叫被中斷、session 過期、上下文壓縮後狀態丟失、媒體傳送失敗、某個外部 API 暫時不通。這些失敗不會出現在華麗短片裡，卻會決定一個 agent 能不能每天被信任。所以 OpenClaw 這類 release 的意義，是把 agent 從「會做事」往「出錯後能恢復」推。

第三個科技重點，是 Strategy 第一次賣出比特幣引發的市場心理變化。CoinDesk 報導，Michael Saylor 的 Strategy 申報賣出三十二枚 BTC，約兩百五十萬美元，用來安排股息資金。這個金額相對它的持倉規模不大，所以看多方會說，這不具實質性，只是資金管理。看空方會說，問題不在三十二枚，而在這是 2022 年以來第一次賣出，象徵意義大於金額。

同一天，crypto funds 出現 2026 年第二大資金流出，但 XRP 和 HYPE 仍逆勢吸金。這代表市場不是全面逃離，而是在重新分配風險。日本執政黨又支持 crypto ETF trading 和日圓穩定幣，顯示制度化入口還在推進。整體來看，crypto 不是只剩投機，也不是已經成熟到沒有風險。它更像正在被傳統金融吸收的一塊高波動資產：有 ETF、有穩定幣、有 tokenized securities 敘事，也有資金流出、監管和治理問題。

LLM 這邊，PyTorch nightly 修了 autograd layout policy 的 state leak，也把 Triton max 和 min dim lowering 做了去重。這類新聞不會上普通使用者的首頁，但它是 AI 基礎設施的日常維修。大型模型訓練和推論之所以能看起來像產品，是因為底層框架每天都在處理這種看不見的狀態、kernel、reduction、layout 問題。它提醒我們，AI 產業不是只有模型發布，還有大量維護成本。

JetBrains 也透過 Hugging Face Blog 推出 Mellum2，一個十二 B Mixture of Experts coding model。這顯示 coding model 的競爭不只在閉源巨頭，也在 IDE 和 developer tool 廠商。JetBrains 的優勢不是單純模型參數，而是它離開發者工作流很近。風險也很清楚：coding model 最後不能只用 benchmark 說服人，還要用真實 repo 任務、延遲、授權和 IDE 整合品質說服人。

中國 AI 商業化也有新訊號。36氪報導，豆包預計6月下旬正式推出付費內容，並加速打通抖音電商。此前 App Store 顯示的訂閱檔位包括每月六十八元、兩百元、五百元。這代表中國 AI 助理正在從免費流量工具轉向商業化。問題是，付費訂閱、電商導流和模型成本三者能不能同時成立。免費時使用者很多，不代表付費時留存一樣漂亮；電商場景能導流，也不代表使用者願意把 AI 助理當成購物入口。

資安今天有一個很典型、也很值得警惕的案例。Krebs on Security 報導，攻擊者濫用 Meta 的 AI support bot 接管 Instagram 帳號，連 Obama White House 和美國 Space Force 高階士官的帳號都曾被置換成親伊朗圖像與訊息。這不是單純帳號密碼外洩，而是支援流程被社交工程化。當平台把 AI support assistant 放進帳號救援流程，攻擊面就從技術漏洞延伸到「如何說服機器幫我重設」。

這對所有公司都是提醒。AI 客服和 AI 支援流程不能只看效率，也要看濫用防線。帳號救援、密碼重設、管理員權限、付款資訊，這些流程一旦被 AI 自動化，就需要更嚴格的身分驗證、人工升級條件和審計紀錄。否則你省下客服成本，可能換來更昂貴的帳號接管事故。

前沿科技還有兩個有意思的方向。量坤科技完成數億元人民幣天使輪和天使加輪融資，把 AI for Science 和量子計算綁在一起，主張高精度資料會限制模型能力上限。這是很大的願景，但短期更像基礎設施押注，因為量子硬體、資料產出和 AI4S 商業化都還在早期。另一條是 OpenAI Robotics 招聘訊號，Sam Altman 說短期專注研發能協助技術工人建設基礎設施的機器人，長期想像個人機器人。這代表 AI 正在從純軟體工作流往實體世界延伸，但硬體供應鏈、場域安全和責任歸屬會讓迭代速度比聊天模型慢很多。

科技延續區簡短帶過：LangChain provider 套件仍在修 cross provider tool call ID 這類 adapter 問題，這提醒多供應商抽象還需要 integration tests；Datasette 1.0 alpha 32 修補 execute write 和 base URL 問題，對 local first 資料工具有實務價值；Simon Willison 的月報把 AI 成本、Anthropic 動能和 Datasette Agent 放在一起看，說明開發者社群正在從模型新鮮感轉向工作流可靠性。

收尾來看，今天的世界和科技其實有同一個問題：誰的承諾能被信任。候選人承諾治安或改革，停火談判承諾降溫，盟友承諾防衛，平台承諾帳號安全，agent 承諾幫你完成長任務。可是承諾真正值錢的時候，不是在順風時，而是在壓力、失敗、質疑和延遲出現時。

所以今天最值得記住的是：系統越大，漂亮的宣告越不夠。你要看的是，它被打斷以後能不能恢復；被質疑以後能不能拿出證據；被要求履約時，能不能真的把承諾交付出來。

今天就到這裡，我們明天繼續。

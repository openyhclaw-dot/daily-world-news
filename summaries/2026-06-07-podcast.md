6月7日，星期日，這裡是今天的世界新聞和科技新聞。今天是第九十五集。這一集的主線很清楚：世界政治這邊，停火、訪問和安全部署都在告訴我們，大國與區域衝突正從聲明進入執行壓力；科技這邊，AI 和加密也不再只是講能力，而是在接受維運、監管、資安和商業化的壓力測試。

先從中東開始。以色列空襲黎巴嫩南部，黎巴嫩軍方和國營媒體說有九人死亡，其中包括三名黎巴嫩軍方人員。這件事最重要的不是單一空襲本身，而是它發生在美方斡旋的新停火安排之後不久。也就是說，紙面上的停火剛被拿出來，地面上的軍事行動就已經在測試它的邊界。

黎巴嫩這邊的說法會強調主權、撤軍和全面停火，認為反覆空襲會破壞任何外交安排。以色列的邏輯則是，只要 Hezbollah 的武裝能力還在，以軍就會保留打擊權。這兩種敘事很難靠一句「雙方克制」解決，因為它們其實對「停火」的定義不同。黎巴嫩想要的是停止攻擊和撤出被占領地區；以色列想要的是邊境威脅被移除。當兩邊對停火的目的都不同，協議就很容易變成各自保留例外的文字。

這也會牽動伊朗戰爭。美國眾議院已經通過限制對伊朗軍事行動的 war powers 措施，這是國會對川普政府中東行動非常明確的反制。但白宮方面已經表態，會在程序上阻擋這項限制案。你可以把它看成兩層戰爭：外面是美國和伊朗，以及以色列、黎巴嫩、Hezbollah 之間的軍事壓力；裡面是美國憲政系統裡，立法部門和行政部門對戰爭權的拉扯。

支持限制軍事行動的人會說，沒有正式宣戰的戰爭不能無限期靠行政權維持；白宮和鷹派會說，中東危機必須保留快速軍事反應空間，不能每一步都等國會授權。這場爭議短期不一定改變戰場，但它會改變美國下一次使用武力的政治成本。尤其在台海、朝鮮半島、紅海這些地方，外界都會觀察：美國總統到底能不能在沒有國會完整授權的情況下，維持長期軍事行動。

第二個重點在東北亞。北韓展示修復後的五千噸級 Kang Kon 驅逐艦海試，金正恩要求海軍要能納入水上與水下核威懾。這個消息和習近平6月8到9日訪問平壤連在一起看，就不是單純軍事展示。北韓是在告訴外界，它的核威懾不只在陸基飛彈，也要往海上延伸；中國則是在這個時間點重新把自己放回北韓外交中心。

習近平七年來第一次去平壤，時間點很微妙。北韓這幾年和俄羅斯越走越近，軍事合作、外交支援、戰爭物資的疑慮都讓北京有壓力。中國過去一直是北韓最重要的後盾，但如果平壤覺得莫斯科也能提供安全和籌碼，北京的影響力就會被稀釋。所以這趟訪問對中國來說，是重新校準東北亞秩序；對北韓來說，是在展示自己不是孤立政權，而是能同時和中國、俄羅斯打交道的核武國家。

首爾和東京看到的會是另一件事。若北韓真的把核威懾往海軍延伸，偵測和攔截難度會增加。過去大家談北韓，多半想到飛彈發射場、移動發射車、地下設施；但如果海上平台也變成威懾的一部分，風險就更分散。這不是明天就會成熟的能力，但它會讓東北亞安全規劃多一層壓力。

台海今天也有相關訊號。荷蘭軍艦 HNLMS De Ruyter 穿越台灣海峽，中國軍方說派出海空兵力跟監。這件事單獨看，可能只是一次航行；但累積起來看，它代表歐洲國家正在用軍艦把自己放進台海議題。北京會說這是外部勢力介入主權問題，荷蘭和歐洲國家的語言則會放在國際法、航行自由和印太存在。

台灣在中間的位置很複雜。外國軍艦通過台海，會讓台灣感覺國際社會沒有放手，但也會讓北京更容易把台灣和外部軍事存在綁在一起，升高中國對台灣周邊的軍事壓力。這就是台海現在的常態：安全訊號本身可能同時帶來安全感和壓力。

美洲今天看兩條。一條是美國世界盃前的安全部署。2026 世界盃很快開踢，這次有四十八隊、一百零四場比賽、十六個城市，跨美國、加拿大、墨西哥。美國端面對的不只是球場安保，還有伊朗戰爭背景、政治暴力、移民執法、AI 假訊息和跨境人流。政府會強調反無人機、AI 攝影機、聯邦和地方協作；民權團體則擔心，移民執法和大規模監控會讓一些球迷處在恐懼氣氛裡。

另一條是哥倫比亞決選。6月21日，Iván Cepeda 和 Abelardo de la Espriella 會進入決選。這場選舉不只是在比左右派，也是在比哥倫比亞要不要延續 Petro 的氣候與能源路線。Cepeda 陣營比較接近現任政府對 Amazon 保護、限制新油氣探勘、反 fracking 的方向；De la Espriella 則得到 Trump 背書，代表更右翼、更重視投資和安全秩序的路線。環保派會把它看成拉美氣候政治的指標；反方會說，哥倫比亞仍需要油氣收入和投資信心。

歐洲這邊，俄羅斯6月2日的大規模空襲仍在延伸。烏克蘭說至少二十二人死亡、一百三十人受傷，Kyiv、Dnipro、Kharkiv 等城市都受波及。NATO 秘書長 Rutte 之後到 Kyiv，防空問題重新被推到前台。這條不是今天才開始的新聞，所以今天只接上最新壓力：烏克蘭需要更多攔截器和反彈道飛彈能力，但歐洲和美國的軍工產能、財政、政治意願，都會決定這些承諾能不能變成真正的彈藥和防空系統。

南亞和東南亞有三條比較值得放在一起看。巴控 Kashmir 警方逮捕數十名 Joint Awami Action Committee 支持者，週二示威前氣氛升高。印度央行維持 repo rate 在百分之五點二五，沒有靠升息硬守盧比，而是用吸引美元流入、非居民印度人美元存款、海外借款避險補貼這些工具。泰國則表示會加入柬埔寨選定的聯合國海上仲裁程序，暫停其他雙邊邊界談判。

這三件事看起來不同，其實都在處理「壓力不要失控」。Kashmir 是社會抗議和地方治理壓力，印度央行是匯率和成長壓力，泰柬仲裁是邊界爭議和軍事誤判壓力。它們沒有像戰爭那樣直接爆炸，但都是政府在試圖把衝突放進制度容器裡。問題是，制度容器夠不夠大，能不能承受街頭、匯率和民族主義同時施壓。

非洲今天要看剛果 Ebola。WHO 和 Africa CDC 推出五點一八億美元、六個月的應變計畫，因為 DRC Bundibugyo Ebola 疫情已被形容成史上第四大，而且 Bundibugyo strain 沒有像 Ebola-Zaire 那樣成熟的專門疫苗和療法。這裡的困難不是只有醫療技術，還有社區信任、交通、衝突地區可及性，以及跨境監測。疫情如果追不上，最後會變成醫療和國家治理能力的雙重壓力。

大洋洲則是 Solomon Islands。新總理 Matthew Wale 表示會檢視2022年和中國簽的秘密安全協議，也要和澳洲談 comprehensive treaty。這代表太平洋島國的安全選擇又一次變成大國競爭的前線。澳洲和美國會看到機會，認為可以降低中國建立軍事存在的風險；北京可能會把它視為外部壓力介入。可是對 Solomon Islands 自己來說，真正核心是怎麼在安全、主權、援助和經濟需求之間保有選擇權。

最後一條世界線，是 WMO 對 El Niño 的警告。6到8月出現 El Niño 的機率達八成，而且可能在下半年延續。這不只是天氣預報，它會影響農業、水資源、公共衛生、能源用電和災害準備。WMO 也提醒，目前沒有證據顯示氣候變遷會增加 El Niño 的頻率或強度，但暖化背景會放大極端事件的衝擊。換句話說，科學預警不是要製造恐慌，而是要讓政府有時間準備。

好，接下來轉到科技。

今天科技第一條，是 Ollama v0.30.7-rc0。這版不是大模型發布，而是把 Windows Hermes 啟動設定改走原生 config path。這聽起來很小，但本地模型工具正在從愛好者工具變成日常基礎設施，跨平台路徑、設定檔位置、版本相容性就會變得很重要。使用者不只想知道模型能不能跑，還想知道下次更新會不會把桌面環境弄壞。

同一條維運線，PyTorch nightly 修了 Inductor 編譯器的兩個細節。一個是不要把 float 乘零直接折成零，另一個是保留 `pin_memory=True` 建構子的 pinned CPU allocation metadata。這不是適合做成華麗標題的新聞，但對訓練和推論效能很重要。AI 工程越走向生產環境，這類底層修補越能決定系統穩定性。

OpenAI Lockdown Mode 今天也值得放進來看。它開始向 eligible personal accounts 和自助 ChatGPT Business 帳號推出，目標是限制可能外送敏感資料的 outbound network requests，降低 prompt injection 最後一段資料外洩風險。這裡要講清楚：Lockdown Mode 不會阻止惡意內容進入上下文，也不會讓 prompt injection 消失。它做的是縮小資料被帶出去的通道。這其實代表 AI 安全開始從「模型不要被騙」轉向「就算模型被騙，系統也不要輕易把資料送出去」。

AI agent 這邊，OpenClaw v2026.6.5-beta.1 修掉 reasoning scaffolding 可能外洩到 QQBot 原生通道的風險，並且補強 MCP tool result 的多種型別處理。CrewAI 1.14.7a2 則加入 conversational flow traces、chat API、route-aware decorators 型別，以及更完整的 LLM event 欄位。這兩個更新放在一起看，代表 agent framework 正在處理一個共同問題：agent 不只是會回答，它還要能被觀測、能被追蹤、能在多工具、多頻道、多模型之間保持邊界。

這裡的風險視角很重要。Agent framework 越像一層作業系統，就越不能把所有錯誤都推給模型。當一個 agent 把不該出現的 reasoning 標記送到聊天頻道，問題可能是輸出清洗；當 tool result 型別不一致，問題可能是 schema；當 conversational flow 沒有 trace，問題可能是可觀測性。這些都是工程問題，不是「換一個更大的模型」就會自動消失。

Simon Willison 的 MicroPython WASM sandbox 也接在同一條線上。他發布 micropython-wasm 0.1a2，並說明如何用 MicroPython 和 WebAssembly 做 Python code sandbox，目標是給 Datasette Agent 一個受限的 code execution 環境。這是 agent 工具化很關鍵的一步：如果你要讓 agent 執行程式碼，就不能只相信它會乖乖做事，你要先把它放在邊界清楚、資源可控、逃逸風險被限制的環境裡。

不過 sandbox 也不是萬靈丹。它會面對套件不足、效能限制、資源控制和長期維護問題。更重要的是，sandbox 只處理執行環境，不處理任務判斷。也就是說，它可以讓危險動作比較難發生，但不能替你判斷哪個任務本身就是不該執行。

加密貨幣今天的情緒很重。CoinDesk 指出，比特幣和以太幣本週跌勢接近 FTX 崩盤以來最糟，整體加密市場市值蒸發約三千九百億美元。今天抓到的最新報價裡，BTC 是6月6日約六萬零五百二十六美元，ETH 約一千五百五十二美元，都是二十四小時內資料，但時間已經超過六小時，所以只能說最新報價，不能說此刻。這裡不需要把它當成短線交易建議，真正值得看的是敘事壓力。

過去幾個月，加密市場有很多制度化故事：ETF、treasury company、stablecoin rails、銀行 digital currency network。可是當價格快速下跌，這些故事會被重新檢驗。Joseph Lubin 相關錢包移動十一萬枚 ETH，以防守二點五九億美元 DAI 債務部位，鏈上分析說更像是降低清算風險，不是直接賣出。看多的人會說，大型持有人主動補抵押品是穩定訊號；看空的人會盯著，如果 ETH 繼續下跌，槓桿部位會不會變成連鎖壓力。

Hyperion DeFi 也要 unwind 二千九百萬美元 HYPE 交易，約八十萬枚 HYPE 回到 treasury，準備改投更有利策略。這聽起來像資本配置，但在下跌市場裡，treasury strategy 很容易被問：你是在主動調整，還是在把流動性風險往後延？同時，美國大型銀行正在建 digital currency network，目標是防堵 stablecoin 造成的存款外流。這代表傳統金融不是沒有看到加密威脅，而是在試圖做自己的版本。

資安今天有一條很值得注意。Meta 證實數千個 Instagram 帳號遭濫用 AI chatbot 相關流程入侵。這件事的重點不是「AI chatbot 很危險」這麼簡單，而是 account recovery 和身份驗證流程一旦接上 AI，錯誤授權會被放大。客服、申訴、帳號救援，這些本來就容易被社工攻擊；如果 AI 讓流程更快，攻擊者也可能更快地測試漏洞。

歐洲還有一條法治與 AI 的交會。英格蘭與威爾斯警方被要求暫停在 court statements 使用 AI。支持者會說，AI 可以幫警察省下大量文書時間；反方會說，法庭文件不只是文字整理，它涉及責任、證據、語氣和可追溯性。如果 AI 改寫了一段證詞，誰負責？如果它省略或重組了關鍵細節，法官和辯方要怎麼查？這些問題都比「AI 會不會寫得比較順」更重要。

前沿科技還有一個比較輕但有意思的工具：ntsc-rs，開源模擬類比電視和 VHS artifacts。它不是主流 AI 新聞，但提醒我們，影像技術不只是在追求更高解析度，也有人在追求可控、可重現的復古失真。這種工具對創作有價值，但不能被誤認為鑑識工具。模擬失真和判斷真實來源，是完全不同的任務。

把今天科技線收束起來，主題其實是「邊界」。Lockdown Mode 是網路外送邊界，OpenClaw 是 reasoning 輸出邊界，CrewAI 是流程觀測邊界，MicroPython WASM 是程式執行邊界，Instagram 帳號事件是身份驗證邊界，警方 court statements 則是法律責任邊界。AI 的能力越強，邊界就越重要，因為能力不是只會創造效率，也會把錯誤放大。

這也是為什麼我不想把今天科技新聞講成一串產品更新。真正的變化是，AI 正在進入那些以前很清楚、現在變模糊的交界處。誰可以代表使用者送出資料？誰可以改寫法律文件？誰可以執行程式碼？誰可以幫忙救回帳號？誰負責清掉模型的內部思考標記？這些問題聽起來不像 demo，但它們會決定 AI 能不能被社會長期接受。

加密也一樣。銀行建自己的 digital currency network，treasury company 調整部位，鏈上大戶補抵押品，這些都不是單純價格故事。它們是在問：當加密資產進入公司資產負債表、銀行存款競爭和監管系統，它還能不能承受傳統金融的風險管理要求。成熟的代價是更多揭露、更多成本、更多被追問的地方。

今天世界新聞和科技新聞合在一起，其實都在講同一件事：承諾要落地，就會遇到執行。停火不是宣布就算，軍艦航行不是通過就結束，世界盃安保不是買設備就完成，Ebola 應變不是有計畫就能追上病毒。AI 也一樣，模型能力不是展示就能進生產，agent 不是會用工具就能被信任，安全模式不是打開就萬無一失。

所以今天最值得帶走的，不是一個單點新聞，而是一個判斷方法。當你看到一個國家、一家公司或一個技術團隊做出很大的承諾，先問它的執行邊界在哪裡：誰監督，誰負責，壞掉時怎麼知道，壞掉後怎麼修。這些問題不華麗，但它們最接近真實世界。

今天就到這裡，我們明天繼續。

## 🧠 LLM

• 🔥13 **PyTorch 回退 Triton autotune fallback 修正** — PyTorch `viable/strict/1779820587` 回退先前的 Triton compile-time autotune fallback 修正，原因是內部 signals 破裂。這類回退看似工程細節，但對 LLM 訓練與推論堆疊很關鍵：compiler path 一改，效能與穩定性都會被放大。風險視角：快速修 compiler bug 的同時，框架團隊必須維持可重現測試，否則下游模型部署會承擔隱性回歸。
  📎 來源：[GitHub](https://github.com/pytorch/pytorch/releases/tag/viable%2Fstrict%2F1779820587)

• 🔥10 **Paul Graham 指創辦人郵件被 AI 文風稀釋** — Paul Graham 的評論被 Simon Willison 摘錄，重點是創辦人郵件若變成一種過度打磨的 AI 新聞腔，反而會降低被讀完的機率。這不是模型能力問題，而是溝通信任問題：LLM 讓文字更順，但也讓真實意圖更難辨識。
